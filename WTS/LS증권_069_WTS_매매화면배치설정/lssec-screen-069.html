단 하나의 매매 화면

거래 전략에 최적화된 레이아웃, 나를 위해 설계하다

WTS의 매매 화면은 각 사용자가 자신의 매매 패턴에 최적화된 화면을 직접 구성할 수 있도록 유연한 선택지를 제공합니다.

먼저 분할 영역의 수와 배치를 선택하고, 분할 영역마다 필요한 메뉴를 추가하거나 삭제할 수 있으며, 자주 사용하는 메뉴를 앞으로 이동시키는 위치 조정도 가능합니다.
분할된 영역 중 잠시 한 곳에 집중하고 싶다면 영역을 크게 확대해서 여러 정보를 여유롭게 확인할 수도 있습니다.
마지막으로 내 마음에 들게 편집한 화면 배치를 저장해두면, 나중에 화면 설정을 불러와 언제 어디서나 동일한 경험으로 매매를 이어갈 수 있습니다.

01 화면 배치 설정

화면배치 선택

  1. 1
    화면배치 설정
    • 투혼 WTS 실행 후, '매매 메뉴 - 우상단 화면 배치 아이콘 - 화면배치 설정'을 선택합니다.
  2. 2
    기본 화면구성 선택
    • 기본적으로 제공되는 7가지 분할화면 중 하나를 선택할 수 있으며, 분할 영역은 편집할 수 있습니다.

자유로운 화면 분할과 배치

02 분할 영역 화면 관리

화면 추가/삭제

  1. 1
    화면 추가/삭제
    • 분할 영역 내 표시되는 화면을 설정하려면 우상단 '더보기 아이콘(…) - 화면 추가/삭제'를 선택합니다.
  2. 2
    화면 선택
    • 화면을 추가하려면 해당 화면 우측에 체크를, 삭제하려면 체크를 해제합니다.
      여기서는 '호가주문', '거래원'에 체크하여 두 개의 화면을 추가해 보겠습니다.
    • 다른 영역에 이미 추가된 메뉴이거나 현재 영역에서 삭제할 수 없는 기본 메뉴인 경우, 체크박스가 비활성화되어 설정을 변경할 수 없으니 유의해 주세요.
  3. 3
    화면 설정 완료
    • 해당 영역의 탭 영역에 새 화면이 추가됩니다.

현재 화면 삭제

  1. 1
    화면 선택
    • 분할 영역 내 삭제하려는 화면을 선택합니다.
  2. 2
    현재 화면 삭제
    • 우상단 '더보기 아이콘(…) - 현재 화면명 삭제'를 선택합니다.
  3. 3
    화면 삭제 완료
    • 조회중이던 화면이 탭에서 삭제됩니다.

메뉴 순서 변경

  1. 1
    탭 이동
    • 순서를 변경하려면 해당 화면의 탭을 변경 위치로 드래그합니다.
  2. 2
    순서 변경 완료
    • 탭 순서를 변경하여 사용 빈도나 우선순위에 따라 화면 순서를 조정할 수 있습니다.
화면 탭이 많을 때 빠르게 정리하기
  • 화면 탭 수가 많아 영역 내 전체 화면명이 보이지 않는 경우, 화살표 아이콘(▼)을 선택하여 표시되는 화면탭 목록 팝업에서 삭제 아이콘(×)으로 탭을 삭제하거나 이동 아이콘(↕)을 드래그하여 탭 순서를 변경할 수 있습니다.

더 자유로운 활용을 위한

03 기타 분할화면 기능

분할 영역 추가

  1. 1
    분할 영역 추가
    • 탭 메뉴를 드래그하여 분할 영역의 상하좌우 밖으로 이동하면 해당 메뉴를 새 분할 영역으로 추가할 수 있습니다.
    • (1-1번) 화면 탭을 분할 영역 우측으로 드래그하면 새 분할 영역이 우측에 추가됩니다.
    • (1-2번) 화면 탭을 분할 영역 하단으로 드래그하면 새 분할 영역이 하단에 추가됩니다.

현재 영역 삭제

  1. 1
    현재 영역 삭제
    • 삭제하려는 분할 영역의 우상단 '더보기 아이콘(…) - 현재 영역 삭제'를 선택합니다.
  2. 2
    분할영역 삭제 완료
    • 해당 분할 영역과 영역 내 화면 탭 전체가 화면에서 삭제됩니다.

분할 영역 전체확대

  1. 1
    영역 전체확대 / 화면축소
    • 영역 우상단의 전체확대 / 화면축소 아이콘으로 특정 영역을 확대하거나 축소할 수 있습니다.

화면배치 저장 / 불러오기

  1. 1
    화면배치 저장
    • 우상단 '화면 배치 아이콘 - 화면배치 저장'으로 직접 편집한 화면배치 및 탭 설정을 저장할 수 있습니다.
  2. 2
    화면배치 불러오기
    • 이전에 저장한 화면배치가 있다면 화면배치 설정 시 해당 정보를 불러올 수 있습니다.
