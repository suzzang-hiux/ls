이런 상황을 패턴화해서, 빠르게 주문할 수 있다면 어떨까요?

[5140] 주식원큐주문 화면은 주문유형, 수량, 가격 등 자주 사용하는 주문 조건을 미리 저장해두고, 필요할 때 클릭 한 번으로 주문을 접수합니다.

번거롭고 복잡한 입력 과정을 줄여주기 때문에 일반 주문보다 빠르게 대응할 수 있고, 시장 변동에 맞춰 준비한 상황별 주문 패턴으로 순간의 기회도 놓치지 않습니다.

한 마디로 주문을 미리 장전해두고, 필요할 때 바로 쏘는 단축키라 할 수 있죠.

01기본 화면 구성

화면은 총 3개 영역으로 나뉘며, 주요 기능인 주문 패턴 영역과 주문을 보조하는 시세 조회 영역, 계좌 정보 조회 영역으로 구성되어 있습니다.

영역 주요 기능
A 시세 조회 영역 종목 시세 정보를 조회할 수 있습니다.
B 주문 패턴 관리 영역 자주 사용하는 주문 패턴을 설정, 저장하고 주문할 수 있습니다.
C 계좌정보 조회 영역 주문 체결내역, 매매손익, 잔고, 예수금 등을 확인할 수 있습니다.

02주문 패턴 설정하기

  1. 1
    설정 팝업 조회
    • 우측 상단의 설정 버튼(1번)을 클릭하면 주문 설정 팝업이 표시됩니다.
  2. 2
    거래소 선택
    • 스마트 / KRX / NXT 중 주문하려는 거래소를 선택합니다.
    • 스마트는 당사가 정한 최선집행기준(SOR)에 따라 KRX와 NXT 중 한 곳으로 주문을 제출합니다.
    • 스마트주문시스템(SOR)에 대한 자세한 설명은 홈페이지에서 확인할 수 있습니다.
  3. 3
    결제방법 선택
    • 현금주문 또는 신용/대출주문 중 선택합니다.
  4. 4
    매매유형 변경 여부 / 주문확인창 보기 여부 선택
    • 매매유형 변경 : 체크 시 정정 주문에서 주문유형을 변경할 수 있습니다.
    • 주문확인창 보기 : 체크 시 매 주문마다 주문확인창을 표시합니다.
  5. 5
    주문 패턴 설정
    • 주문 구분, 주문유형, 주문가격, 주문수량을 차례대로 설정합니다.
    • 설정한 패턴의 특징은 요약 문구(5-1번)에서 확인할 수 있습니다.
  6. 6
    메모 설정
    • 설정한 주문 패턴에 메모를 작성하여 별칭으로 활용할 수 있습니다.
    • 메모를 입력하지 않은 경우, 주문 조건이 요약되어 패턴 이름(5-1번)으로 자동 표기됩니다.
  7. 7
    주문 패턴 추가
    • '조건등록' 버튼 클릭 시 주문 패턴 목록(8번)에 패턴을 새로 추가합니다.
  8. 8
    주문 패턴 목록
    • 등록한 주문 패턴을 조회합니다.
    • 패턴 선택 후 설정(5번)에서 수정사항을 저장(8-1번)하거나 삭제(8-2번)할 수 있습니다.
주식 주문유형 정리
보통(지정가) 지정한 가격으로 주문
시장가 주문단가를 지정하지 않고 시장에서 형성되는 가격으로 자동 지정
중간가 최우선 매수호가와 최우선 매도호가의 중간 가격
시간외단일가 정규장 종료 후 10분 단위로 단일가 매매 (16:00~18:00)
조건부지정가 지정한 가격으로 주문 후, 정규장 종료까지 미체결 시 종가로 자동 전환되어 주문 체결
최유리지정가 상대 1차 호가에 지정
최우선지정가 동일 방향 1차 호가에 지정
장전 시간외 정규장 시작 전 시간외 거래 (08:30~08:40)
장후 시간외 정규장 종료 후 시간외 거래 (15:40~16:00)
NXT종가 KRX의 당일 정규장 종가를 기준으로 15:30~16:00 사이 단일가 매매
FOK / IOC 주문조건이란?
  • FOK(Fill-Or-Kill)은 모든 수량을 체결할 수 있을 때만 주문을 체결하고, 단 1주라도 체결할 수 없으면 주문 전체를 취소합니다.
    예) 50주 주문 시 40주만 체결 가능한 경우, 50주 주문 전체를 취소합니다.
  • IOC(Immediate-Or-Cancel)은 체결 가능한 수량만큼 체결하고, 체결되지 않은 잔량은 취소합니다.
    예) 50주 주문 시 40주만 체결 가능한 경우, 40주를 체결하고 10주는 취소합니다.
주의하세요!
  • '주문확인창 보기' 체크를 해제하면 확인 절차를 생략하기 때문에, 되돌릴 수 없는 주문 실수가 발생할 수 있습니다. 이 점에 유의하여 신중히 설정해 주세요.

03패턴 주문하기

  1. 1
    거래소 선택
    • 패턴 등록 시 지정한 거래소에 해당하는 패턴 목록을 조회할 수 있습니다.
  2. 2
    종목 선택
    • 주문하려는 종목을 선택합니다.
  3. 3
    패턴 주문하기
    • 설정해둔 패턴을 클릭하여 주문을 접수합니다.
    • '주문확인창 보기' 설정 시 주문확인창(3-1번)이 표시됩니다.
내가 설정한 패턴을 목록에서 인지하기 어렵다면
  • 메모에 짧은 별칭을 등록하면 세로 목록에서도 인지하기 쉽습니다.
  • '가로' 버튼을 클릭하면 패턴 목록이 가로로 표시되어, 긴 명칭의 패턴도 원활하게 확인할 수 있습니다.

주문을 더 빠르고 편리하게

04기타 기능

주문패턴 독립 실행

  1. 1
    주문 패턴 단독 사용
    • 특정 패턴을 화면 밖으로 드래그하여 단독 팝업으로 사용할 수 있으며, 다른 화면과 조합하여 종목 정보를 연동하여 주문할 때 활용할 수 있습니다.
  2. 2
    주문 패널 팝업
    • 독립된 주문패턴 위에는 주문 대상 종목명(2-1번)과 주문 계좌(2-2번)이 표시됩니다.

주문 내역 조회

  • 미체결|잔고, 미체결, 잔고, 체결확인 탭 :

    미체결 주문 내역을 조회하고 정정/취소하거나, 실시간 잔고를 확인할 수 있습니다.

  • 예수금 탭 : 계좌별 예수금 현황을 확인할 수 있습니다.
  • 당일매매, 매도손익 탭 : 일별 매매 내역 정산 결과를 확인할 수 있습니다.
  • 주문가능 탭 : 종목별 증거금률 및 주문가능금액 등을 확인할 수 있습니다.
  • 챠트 탭 : 종목 · 업종 · 상품별 차트를 확인할 수 있습니다.

지금까지 주문 패턴을 미리 등록하여 효율적이고 빠르게 주문하는 방법을 살펴보았습니다.
