[5556]주식빠른호가주문2 화면에서 ‘설정’ 버튼을 클릭하여 팝업에서 다양한 기능을 설정할 수 있습니다.
소개해드리는 여러 기능을 둘러보고 본인에게 맞는 설정으로 화면을 바꾸어 보세요.

손이 눈보다 빠르게

01주문 속도 높이기

빠른 주문수량 · 금액 설정하기

설정 화면에서 종목 조회 시 자동 입력할 주문수량을 미리 설정할 수 있습니다.

  1. 1
    매도 시 실시간 수량 계산
    • 선택 시, 매도 수량과 매수 수량 입력란이 분리(1-1번)되고, 매도 주문을 접수하는 즉시 실시간으로 계산된 주문가능 수량이 입력란에 자동 표시됩니다.
    • '가' 버튼(1-2번) 클릭 시 주문가능 수량이 설정한 비율(1-3번)만큼 주문수량 입력란에 표시됩니다.
  2. 2
    주문가능 수량 설정
    • 선택 시 주문수량 입력란(2-1번)에서 매도 수량과 매수 수량을 통합 관리합니다.
    • 현재가와 상한가 중 어떤 가격을 기준으로 주문 수량을 계산할지 선택할 수 있습니다.
  3. 3
    주문가능 수량 계산 방법 설정

    • 보유잔고가 없는 경우 :

      주문금액(3-3번)과 기준가(3-1번)를 바탕으로 매수가능 수량이 자동으로 계산되어 주문수량 입력란(2-1번)에 표시됩니다. 또한, 종목 선택 시 주문가능 수량(3-2번)에 설정된 비율(%)에 따라 수량을 조절할 수 있습니다.

    1) ’종목선택 시 주문가능 수량’(3-2번) 체크 해제 시 100%를 기본값으로 계산합니다.

    2) ’설정금액’(3-3번) 체크 해제 시 주문가능금액을 기본값으로 계산합니다.

    • 보유잔고가 있는 경우 :

      계산 방법 체크 여부와 무관하게 매도가능 수량을 우선하여 주문수량 입력란(2-1번)에 표시합니다.

    • ’잔고 청산가능수량의 % 우선 연동’(3-4번) 체크 시 청산가능수량 비율을 계산합니다.
    • ‘호가 더블클릭 후 주문가능수량 계산 표시’(3-5번) 체크 시 호가를 더블클릭하면 매수 가능수량이 주문수량 입력란(2-1번)에 표시됩니다.
  4. 4
    빠른수량 설정
    • 클릭하면 주문 수량 입력란에 자동 입력되는 주문 수량 또는 주문 비율을 미리 3개까지 설정할 수 있습니다.
    • 주식수(주), 비율(%), 금액기준으로 설정 시 호가 화면에 (4-1번)과 같이 설정한 수량이 반영됩니다.
매도 시 실시간 수량 계산 vs 주문가능 수량
  • 매도 시 실시간 수량 계산은 매도 주문을 접수하는 즉시 실시간으로 주문가능 수량을 계산합니다.
    별도의 잔고 확인 없이 다음 주문을 빠르게 접수할 수 있습니다.
    보유수량이 100주인 경우 주문수량에 10주를 입력하고 매도 주문을 접수하면, 주문 직후 주문수량 입력란에 매도가능 수량인 90주가 입력됩니다.
  • 지정수량 반복주문은 입력한 주문수량이 유지되어 동일한 수량을 분할 주문할 때 유리합니다.
    주문수량에 10주를 입력하고 매도 주문을 접수하면, 주문 직후에도 기존에 입력한 주문수량 10주가 주문수량 입력란에 그대로 유지됩니다.

주문 실행 방식 설정

마우스나 키보드로 주문을 접수하는 방식을 설정합니다.

  1. 1
    주문 실행 방식 설정
    • 호가 영역(1-1번, 1-2번)에서 주문 접수 방식을 선택할 수 있습니다.
      더블클릭 주문(기본 설정) : 마우스를 두 번 연속으로 클릭해 주문을 제출합니다.
      원클릭 주문 : 마우스를 한 번만 클릭해 주문을 제출합니다.
  2. 2
    키보드 주문 기능
    • 키보드 단축키를 활용한 주문이 가능합니다.
    항목 단축키
    매수주문 Enter
    매도주문 Enter
    일괄매도 Ctrl + Enter
    정정주문 Shift (선택) → 이동 → Shift + Enter (주문)
    취소주문 Delete
    전량취소 Shift + Delete
    매수/매도 전량취소 Ctrl + Delete
    매도가능 Ctrl + Q
    매수가능 Ctrl + W
    가능금액 Ctrl + A

수량 초기화 설정하기

주문 직후 주문 수량 입력란을 초기화할지 선택할 수 있습니다.

  1. 1
    주문 후 수량 지움
    • 체크 시 주문 접수 후 주문수량 입력란(1-1번)이 ‘0’으로 초기화됩니다.
    • 체크 해제 시 주문 접수 후에도 주문수량 입력란(1-1번)이 초기되지 않고 직전 주문수량을 유지합니다.

시장 흐름을 빠르게 읽는

02화면 표시 설정

호가등락률 표시

호가창 측면의 등락률 칼럼과 표시 기준을 설정합니다.

  1. 1
    호가등락률 칼럼 추가
    • 호가창 우측에 등락률 칼럼(1-1번)을 추가합니다.
    • 기준가대비, 시가 / 고가 / 저가 대비, 평균매입가 / 손익분기매입가 대비 등 다양한 등락률 기준을 설정할 수 있습니다.

색상강조 표시 · 호가잔량 그래프 보기

호가창에 다양한 기준으로 색상 및 도형 표시를 적용할 수 있습니다.

  1. 1
    색상설정
    • 현재가 외곽선 표시(1-1번) : 현재가 주변을 붉은색 테두리로 강조합니다.
    • 현재가 배경색(1-1번) : 현재가 배경을 지정색(예시 : 노란색)으로 표시합니다.
    • 호가 배경(1-2번) : 호가 목록 배경을 지정색(예시 : 흰색)으로 표시합니다.
    • 손익분기 매입가(1-3번) : 손익분기 매입가 배경을 지정색(예시 : 빨간색)으로 표시합니다.
    • 평균매입가, 미체결주문가 도형표시 (1-4, 1-5, 1-6번) : 평균매입가에 도형(●), 미체결주문이 있는 호가에 매수/매도 여부에 따라 빨간색, 파란색 도형(◆)을 표시합니다.
  2. 2
    정적 VI 발동 예상가 표시
    • 정적 VI 발동 예상가 배경을 지정색(예시 : 주황색)으로 표시합니다.

호가잔량 그래프 · 체결량 색상 표시 · 호가 메모 보기

호가잔량, 체결량, 시세 등을 시각화하여 볼 수 있습니다.

  1. 1
    화면 고정
    • ‘고정‘ 버튼으로 호가창을 고정하면 호가잔량 그래프와 체결량 표시설정을 적용할 수 있습니다.
  2. 2
    호가잔량 그래프 보기
    • 체크 시 매도 호가잔량 영역(2-1번)과 매수 호가잔량 영역(2-2번)에 막대그래프가 표시됩니다.
  3. 3
    체결량 표시설정 적용
    • 체크 시 일정량 이상의 체결량(3-1번) 내역이 발생하면 숫자와 배경색을 강조 표시합니다.
    • ‘체결량 표시설정‘ 버튼으로 체결량 표시 조건과 표시 방식을 설정할 수 있습니다.
  4. 4
    호가 메모 보기
    • 당일 시가, 고가, 저가, 가중 평균가 등에 해당하는 가격을 호가창 우측에 색상으로 구분표시(4-1)해주며, 색상 표시 위에 마우스를 올리면 항목에 대한 설명을 툴팁으로 확인할 수 있습니다.
주의하세요!
  • 변동성이 크거나 거래량이 많은 종목 조회 시 ‘호가잔량 그래프 보기’ 또는 ‘체결량 표시설정 적용’ 기능을 사용하면 처리해야 할 데이터가 많아 변동사항을 실시간으로 반영하지 못하거나 HTS에 부하를 줄 수 있습니다.

기타 사용자 설정

  1. 1
    마우스 휠 이동범위 설정
    • 호가창에서 마우스 휠 상하 이동 시, 한 번에 움직이는 호가 단위를 설정할 수 있습니다.
    • 기본값은 ‘1’ → 휠 한 번에 1호가씩 이동하며, ‘2’로 설정하면 휠 한 번에 2호가씩 이동합니다.
  2. 2
    전체보기시 체결량 / 차트 보기
    • 화면 고정 해제 시에도 체결량, 차트를 확인할 수 있습니다.
  3. 3
    하단 미니차트 보기
    • 화면 하단 미니차트 표시 여부를 설정할 수 있습니다.
  4. 4
    주문에러 메시지 보기
    • 주문이 접수되지 못했을 때 오류 메시지를 팝업으로 표시합니다.
  5. 5
    화면 비율 늘리기
    • 체크 시 화면 모서리를 드래그해 크기를 늘릴 수 있습니다.
  6. 6
    가운데 정렬방식 설정
    • 호가 중앙 위치 값을 설정합니다. 체크 시 매도1호가 기준, 체크 해제 시 매수1호가 기준으로 정렬됩니다.
  7. 7
    종목변경 시 포커스 종목코드 유지
    • 종목 변경 시 커서가 종목코드에 위치해 즉시 다른 종목을 조회할 수 있습니다.
  8. 8
    기타가격 사용
    • 체크 시 호가창 상 · 하단(8-1번)에 직접 설정한 호가가 표시되고, 매수/매도 호가창으로 주문할 수 있습니다.
