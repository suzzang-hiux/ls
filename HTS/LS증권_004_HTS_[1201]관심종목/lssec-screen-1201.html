투자 전략에 따른 관심 종목의 그룹관리

주식시장은 매일 변화하며, 테마 종목도 그 흐름에 따라 빠르게 교체됩니다.
이러한 흐름을 놓치지 않기 위해서는 관심종목 화면이 필수적입니다.

[1201] 관심종목 화면은 관심종목을 등록하고 실시간 시세와 관련 정보를 조회하는 기능입니다.
주식, 선물, 지수 등 다양한 자산을 그룹별로 관리하며 빠르게 확인할 수 있습니다.

투자 상품별 그룹 관리 및 다양한 정보 항목 편집 기능!

이 화면이 필요한 이유!

기능요소 이런 점이 좋아요
투자 상품별 그룹핑 주식, 선물옵션, ELW, 업종별 지수 등을 별도 구분하여 그룹핑 관리
다양한 정보 편집 관리 시가, 고가, 저가, 매입단가, 평가금액, 수익률 등 다양한 필드 편집 기능
주요 종목 관리 북마크 표시, 투자 아이디어 메모로 더 효율적인 종목 관리
실시간 데이터 변화 인식 등록된 종목의 데이터 변화시 색상 또는 알람을 통해 변동 상황 제공

내가 고른 관심 종목을 담을 바구니

01관심 그룹 만들기

  1. 1
    새 관심 그룹 추가
    • 관심 그룹을 생성하기 위해서는 새로운 그룹명을 입력(1-1번)하거나 유형화 되어 있는 관심 그룹 리스트(1-2번)를 선택하여 관심 그룹으로 등록할 수 있습니다.
    • 1개 그룹 당 최대 100개의 관심종목을 등록할 수 있습니다.
관심그룹을 한 화면에서 여러 개 보고 싶을 때는?
  • 분할 버튼(2번) 선택 후 나타나는 설정창에서 원하는 만큼의 개수와 배치 방법을 선택하면 됩니다.

마음에 든 종목을 '찜'하는 네 가지 방법

02관심 종목 등록하기

‘설정’ 화면에서 관심종목 등록하는 방법

  1. 1
    관심종목 등록을 위한 ‘설정‘ 버튼 클릭
    • 클릭 시 ‘관심종목설정’ 팝업 화면이 나타납니다.
  2. 2
    관심종목설정
    • 관심그룹을 선택(2-1번)한 후, 좌측(2-2번) 영역의 종목 리스트 중 등록할 관심종목을 선택하고 ‘추가’ 버튼을 클릭하면 우측(2-3번) 영역에 해당 종목이 추가됩니다.

유형화된 '종목군'을 관심종목으로 등록하는 방법

  1. 1
    유형별 관심종목군 선택
    • 상승률 상위, 테마별, 외인 / 기관 거래 상위 등 다양한 유형의 관심종목군을 선택하여 추가할 수 있습니다.
    • 관심 종목군 유형을 선택한 경우, 해당 유형의 명칭이 관심그룹명으로 자동 반영됩니다.

목록에서 관심종목을 바로 등록하는 방법

  1. 1
    관심그룹 화면에서 마우스 우클릭
    • 마우스 우클릭 시 팝업(1-1번)에서 종목 추가, 삽입, 삭제, 변경, 빈줄 추가 기능을 사용할 수 있습니다.
      단, 종목 삽입, 삭제, 변경, 빈줄 추가 기능은 관심종목 리스트 위에서만 활성화됩니다.
  2. 2
    종목 검색창
    • 원하는 종목을 검색하여 바로 관심종목으로 등록할 수 있습니다.

종목 검색창에서 관심종목을 등록하는 방법

  1. 1
    종목 검색 영역 조회
    • 버튼 클릭 시 표시되는 검색 영역(1-1번)에서 종목을 검색하여 관심종목으로 등록할 수 있습니다.
    • '영문종목코드 검색시'에 체크하면 종목코드 첫 자리가 영문으로 시작하는 종목이 검색결과에 표시됩니다.
추가 버튼과 삽입 버튼의 기능 차이
  • 추가 버튼 : 관심 종목군 최하단에 등록됩니다.

  • 삽입 버튼 :

    관심 종목군 사이에 삽입되며, 특정 종목 클릭 후 삽입 버튼 실행시 해당 종목의 상단에 등록됩니다.

관심종목 200% 활용하기

03필드 유형 및 거래소 선택 및 기타기능

  1. 1
    필드 유형 선택
    • 선택한 필드에 따라 현재가, 시가, 증거금률, 일봉차트 등 각기 다르게 설정된 필드(항목) 구성을 조회할 수 있습니다.
    • 원하는 필드(항목)을 직접 설정하려면, 설정 버튼(4번) 클릭 후 '관심종목설정' 창의 '필드편집' 탭에서 필드를 편집하고 저장하면 됩니다.
  2. 2
    거래소 유형 선택
    • 통합(KRX + NXT 통합 시세) / KRX(KRX 시세) / NXT(NXT 시세) 3가지 유형으로 구분되어 시세 제공 거래소를 선택할 수 있습니다.
  3. 3
    예상가
    • 버튼 클릭 시 필드 영역내에 예상체결가 및 예상대비 항목이 컬럼에 추가됩니다.
  4. 4
    설정
    • 관심 종목 화면의 필드 항목을 다양한 방법으로 편집할 수 있습니다.
  5. 5
    PC 저장
    • 사용자가 구성한 관심종목 필드 조합을 현재 사용중인 PC에 저장하는 기능이며 재접속 시에도 동일한 화면 상태로 불러올 수 있습니다.
      다만, 이 저장 기능은 해당 PC에만 적용되며, 다른 PC 또는 기기에서는 동일 구성을 불러올 수 없습니다.
  6. 6
    복수차트
    • 관심 종목에 등록된 복수 종목의 차트를 동시에 비교할 수 있는 기능입니다.
  7. 7
    해외관심 가져오기
    • 해외주식을 국내주식과 동일한 그룹에 관심종목으로 등록합니다.

04설정 기능

관심그룹관리

관심 그룹 화면은 다양한 형태로 사용자의 취향에 맞도록 변경이 가능합니다.
관심 그룹관리 / 필드 편집 / 기타설정 / 알람설정 총 4가지로 구분되어 있는데 차근차근 하나씩 살펴보겠습니다.

  1. 1

    종목 리스트 :

    관심 그룹에 추가할 종목을 검색하는 영역입니다.
    종목 리스트 검색 또는 종목명 / 종목코드 / 가나다순 등 다양한 방법으로 종목검색이 가능합니다.

  2. 2

    종목 추가 / 삭제 버튼 : 전체 종목 일괄 추가 / 삭제 및 종목별 추가 / 삭제 가능합니다.

  3. 3

    필드그룹 연동 : 관심 그룹 변경시에 이전의 필드 정보가 그대로 적용되는 기능입니다.

  4. 4

    추가 / 수정 / 삭제 : 관심 그룹을 추가 또는 삭제 할 수 있으며, 관심 그룹명을 수정할 수도 있습니다.

  5. 5

    복사 : 등록된 관심 그룹 선택 후 '복사' 버튼 클릭 시 선택한 관심그룹과 동일한 그룹이 생성됩니다.

  6. 6

    위치 이동 : 관심 그룹의 위치를 앞뒤로 이동 할 수 있습니다.

  7. 7

    등록된 관심 종목의 리스트가 나타나는 화면이며 클릭 시 엑셀에 저장한 종목도 불러올 수 있습니다. (11번)
    또한 잔고불러오기 버튼(7-2번) 클릭 시에는 보유종목의 수량, 매입단가, 평가금액 수익률 등 다양한 정보를 함께 불러올 수 있습니다.

  8. 8

    관심 그룹내 종목 검색/삭제 : 종목검색 시 해당 종목이 등록된 관심그룹명을 표시해줍니다.

  9. 9

    [2002] 해외주식 관심종목에 등록된 관심그룹을 [1201] 관심종목 으로 가져오는 기능입니다.
    종목리스트에서 ‘해외주식관심종목‘ 선택 시 그룹이동 버튼이 활성화됩니다.

  10. 10

    서버에 저장하기(전체) : 다른 기기에서도 동일한 데이터를 조회할 수 있도록 서버에 편집 내용을 저장합니다.

서버 저장하기 vs PC 저장하기
  • 서버 저장하기(10번) : 아이디 기준으로 저장하므로 다른 PC에서도 저장한 데이터를 그대로 불러올 수 있는 매우 유용한 기능입니다. ‘PC 저장하기’는 특정 PC에만 저장하며 해당 PC에서만 데이터를 불러올 수 있다는 점에서 차이가 있습니다.
  • PC 저장하기 : 특정 PC에만 저장하며 해당 PC에서만 데이터를 불러올 수 있습니다.
  1. 1

    유형별 필드 구성 항목 :

    관심종목의 다양한 정보를 선택할 수 있는 필드 구성 항목이 유형별로 분류되어 있습니다.

  2. ※ 유형별 분류 :

    전체 / 시세 / ELW / 시간외단일가 / 재무 / 기술적 분석 / 외국인 / 투자자별 매매동향 / 매입단가 / 기타

  3. 2

    현재 적용되어있는 필드명 리스트입니다.

  4. 3

    필드명 추가 / 삭제 / 수정 가능합니다.

  5. 4

    선택한 필드항목을 한 개씩 등록 / 삭제 합니다.

  6. 5

    필드항목 전체를 등록 / 삭제 합니다.

  7. 6

    등록된 필드항목의 순서를 변경합니다.

  8. 7

    초기화 : 클릭 시 ‘선택된 필드그룹 초기화’ or ‘전체 필드그룹 초기화’ 선택 가능합니다.

  9. 8

    기본테마색 사용 : 체크 시 코스피 종목명은 파란색, 코스닥 종목명은 빨간색으로 표시됩니다.

이번에는 관심그룹에 등록된 필드의 영역을 다양한 방법으로 변경해보겠습니다.

예를 들어 볼까요?

필드의 3행과 2행까지의 항목을 고정하려는 경우

  • 아래의 화면에서 1번에서 고정행을 3으로 선택하고, 고정열을 2로 선택하면 1-1번과 같이 3행 고정 2열 고정이 됩니다. 좌우 스크롤바 또는 상하 스크롤바를 이동시키면 고정된 영역은 움직이지 않고 다른 영역만 움직이는 것을 확인할 수 있습니다.
  1. 1

    고정행/고정열 : 필드 영역의 행과 열을 고정시킬 수 있습니다.

  2. 2

    행높이 : 필드 영역의 행간 높이를 변경할 수 있습니다.

  3. 3

    종목/메모굵게 : 체크 시 텍스트는 굵은 글씨로 반영됩니다.

  4. 4

    대비필드 모양 : 현재가의 증감표시 유형을 변경합니다.

  5. 5

    예상가격 필드 :

    예상체결가, 예상대비, 예상등락률, 예상매수잔량 등 예상 가격과 관련된 항목을 필드에 추가할 수 있습니다. 위치 선택란의 숫자는 필드항목의 몇번째에 보여줄 것인지 선택하는 기능입니다.

  6. 6

    보유종목 실시간 연동 :

    필드 영역에 있는 보유 종목에 대하여 매입수량, 매입단가, 평가금액, 수익률 등 항목이 실시간으로 반영됩니다.

기타설정

  1. 1

    데이터 변환 :

    실시간 데이터 수신 시, 해당 종목의 데이터 변화 색상을 행단위 또는 셀단위로 구분하여 적용할 수 있습니다.

  2. 2

    더블클릭시 연결화면 :

    관심종목 화면에서 투자상품 유형별로 클릭 시 이동하는 화면을 지정할 수 있습니다.

  3. 3

    확인 메세지 보기 :

    종목삭제 / 추가 / 변경내용 저장 / 변경내용 저장 시 확인 메세지 팝업 생성 여부를 선택합니다.

  4. 4

    배경색 지정 : 필드 영역의 배경색상을 변경할 수 있습니다.

  1. 5

    관심그룹 이동버튼 보기 :

    체크 시 (5-1번) 영역에 등록한 관심 그룹명을 구분하는 버튼이 표시됩니다.

  2. 6

    종목명/현재가 클릭시만 종목연동 :

    종목명과 현재가의 컬럼을 클릭할 경우에만 화면간 연동 기능이 적용됩니다.

  3. 7

    평가손익/평가금액 보기 :

    보유종목을 관심종목으로 등록한 경우, (7-1번) 영역에 해당 종목의 평가손익 및 평가금액이 표시됩니다.

  4. 8

    관심그룹 탭 간격 유지 : 관심그룹 탭 간격(5-1번)을 유지할 것인지 선택합니다.

  5. 9

    빈줄 정렬 :

    행간에 빈줄 삽입 후 메모 글을 입력하는 경우(9-1번) 해당 메모 글의 위치를 좌측 / 중간 / 우측으로 변경할 수 있습니다.

알람설정

  1. 1

    종목 시그널 및 시세 알람 표시 :

    각 항목별 체크 시, 설정된 조건에 부합하는 알람과 신호가 컬럼(2번) 영역에 표시됩니다.

  2. 2
