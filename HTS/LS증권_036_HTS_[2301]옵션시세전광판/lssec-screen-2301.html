시세 조회와 주문을 동시에, 옵션 거래의 중심 창구

옵션은 기초자산 가격 변동에 따라 가치가 달라지는 파생상품으로, 소액으로도 레버리지 효과를 얻을 수 있지만 변동성이 크기 때문에 신속한 시세 파악과 주문 기능이 중요합니다.

[2301] 옵션시세전광판은 옵션 전 종목의 시세 정보를 사용자 맞춤 항목으로 조회하고, 필요에 따라 호가창과 주문창을 펼쳐 빠르게 매매할 수 있는 화면입니다.

조회와 주문을 연결하는,

이 화면이 옵션의 핵심 무대인 이유!

특징 이런 점이 좋아요
시세 / 지표 통합 조회 현재가, 거래량, 미결제약정, 내재변동성 등 옵션 주요 지표를 한눈에 확인
맞춤형 편집 기능 관심 항목을 직접 선택해 자신만의 시세 전광판을 구성, 효율적인 매매 환경 제공
호가 / 주문 동시 실행 호가창을 보며 매수·매도 주문창을 즉시 열어 신속하게 매매 가능

    옵션 종목 선택

  2. 3

조회 항목 설정

  1. 1
    항목 설정
    • ‘설정’ 버튼에서 그룹별 조회 항목을 직접 편집하여 최대 8개의 그룹까지 저장할 수 있습니다.
    • 그룹(1-1번) 선택 후 항목을 편집하면 (1-2번)과 같이 화면에 조회 항목이 반영됩니다.
옵션 주요 용어

  • 내재변동성(IV) :

    시장에서 거래되는 파생상품가격을 통해 옵션이론 가격결정모형인 블랙숄즈모형을 통해 역추적된 변동성

  • 대표 I.V :

    동일월물 각 내재변동성의 거래량 가중평균으로, 거래량이 많을수록 시장가격의 내재변동성 예측력이 높아집니다.

  • 역사적변동성(HV) : 일정기간 동안 종가 데이터를 가지고 계산한 변동성입니다.

  • HV(90) :

    과거 90일 동안 기초자산의 역사적 변동성으로, 과거 90일간 종가의 표준편차로 계산(일단위표준편차 X √90)합니다.

  • 옵션잔존일 :

    옵션 계약의 만기일까지 남은 기간을 의미하며, 만기일이 지나면 옵션의 권리는 소멸됩니다.

  • 이론가 :

    종목을 옵션이론 가격결정모형인 블랙숄즈모형을 적용해 계산한 이론적인 가격으로, 실제 시장가격과 비교하면 고평가 · 저평가 여부를 판단할 수 있음

  • 행사가 :

    옵션의 만기 시점에 행사할 수 있는 가격이며, KOSPI 200의 현재가에 가장 근접한 가격의 배경색이 보라색으로 표시됩니다.

  • 내재가치 : 옵션을 즉시 행사할 경우 얻을 수 있는 실질적 이익으로, 항상 0이상의 값을 가짐
  • 시간가치 : 옵션 만기일까지 기초자산의 가격 변동으로 상승 가능한 옵션 가치에 대한 기대치
  • 미결제 약정 : 현재까지 반대매매로 청산되지 않은 계약수

민감도 설정

  1. 1
    민감도 설정
    • ‘민감’ 버튼에서 민감도 지표의 계산 방법 및 기준을 설정할 수 있습니다.
번호 항목 설명
1-1 민감도 설정 사용 민감도 사용 여부를 선택할 수 있습니다.
1-2 잔존일수 계산방식 옵션 만기일까지 남은 일수를 계산할 때 휴일 포함 여부, 일자 기준, 시간 기준, 최종거래일 포함 여부 등 계산 기준을 세세하게 설정할 수 있습니다.
1-3 기초자산 선물비율 민감도 계산 시 적용할 기초자산의 반영 비율을 지정합니다. (기본값 100%)
1-4 변동성 설정 민감도 계산에 반영할 변동성 지표를 선택합니다.
HV(과거 가격을 기준으로 한 역사적 변동성), IV(옵션 프리미엄에서 산출한 내재 변동성), 사용자 입력값 중 선택할 수 있습니다.
1-5 대표 내재변동성 설정기준 (1-4번) 변동성 설정(%)에서 '대표 I.V' 선택 시 활성화됩니다.
옵션의 내재변동성 계산 기준을 거래량 가중, 거래대금 가중 중 선택할 수 있습니다.
민감도 지표 설명
  • 델타 : 기초자산 가격 한 단위 변동에 대한 옵션가치의 변동
  • 감마 : 기초자산 가격 한 단위 변동에 대한 옵션델타 변동
  • 쎄타 : 잔존일수 1일 경과에 대한 옵션가치의 변동
  • 베가 : 변동성 1%변동에 대한 옵션가치의 변동
  • 로우 : 이자율 1%변동에 대한 옵션가치의 변동

지수 환산값 표시

  1. 1
    지수환산값 표시
    • 체크 시 각 행사가를 종합지수로 환산한 값(1-1번)이 추가로 표시되며, 종합지수 및 KOSPI200이 변경될 때마다 자동으로 변경됩니다.
    • 지수환산 계산식 : 지수환산값 = (현재 종합지수 × 행사가격) ÷ KOSPI200

02옵션 주문 모드

매수 · 매도 · 정정 · 취소 주문

  1. 1
    전용주문 / 전광판 전환 버튼
    • 버튼 클릭 시 주문창을 하단에 표시하는 옵션 주문 모드와 시세 전광판을 확인하는 시세 조회 모드를 전환할 수 있습니다.
  2. 2
    매수 · 매도 · 정정 · 취소 주문 선택
    • 매수 / 매도 주문 영역(2-1번) 클릭 시 선택 영역에 아이콘(●)이 표시되며 주문 탭(2-2번)이 해당 주문창으로 전환됩니다.
  3. 3
    주문유형 선택
    • 유형번호를 직접 입력하거나 드롭다운에서 주문유형을 선택할 수 있습니다.
  4. 4
    주문수량 입력
    • 수량을 직접 입력하거나 ‘수’ 버튼 클릭 시 표시되는 팝업에서 설정된 수량을 선택할 수 있습니다.
    • ‘총가능’ 버튼 클릭 시 매수가능 수량이 자동 계산되어 입력됩니다.
    • ‘신규’ 버튼 클릭 시 신규 주문 가능한 계약수가, '청산' 클릭 시 청산 가능한 보유잔고가 입력됩니다.
    • 수량 입력 버튼(4-1번) 클릭 시 해당 수량이 누적 입력됩니다.
      (예 : 매수 주문창에서 수량 '5' 와 '20' 버튼 클릭 시 '25'가 입력)
  5. 5
    주문가격 입력
    • 가격을 직접 입력하거나 ‘호’ 버튼을 클릭하여 표시되는 호가창에서 선택할 수 있습니다.
    • 호가창(5-1번)에서 호가 클릭 시 해당 호가가 자동으로 입력됩니다.
  6. 6
    동시주문 버튼
    • 클릭 한 번에 (2-2번)영역의 두 주문이 좌측부터 우측 순으로 동시에 접수됩니다.
  7. 7
    설정
    • 초기 입력값, 주문확인창, 빠른 주문수량 버튼 등 선물/옵션 주문 환경을 설정할 수 있습니다.
주식 주문유형 정리
지정가 지정한 가격으로 주문
시장가 주문단가를 지정하지 않고 시장에서 형성되는 가격으로 자동 지정
조건부 장마감까지 지정가에 주문이 체결되지 않을 경우, 시장가로 주문가격이 전환되어 체결
최유리지정가 상대 1차 호가에 지정
FOK / IOC 주문조건이란?
  • FOK(Fill-Or-Kill)은 모든 수량을 체결할 수 있을 때만 주문을 체결하고, 단 1주라도 체결할 수 없으면 주문 전체를 취소합니다.
    예) 50주 주문 시 40주만 체결 가능한 경우, 50주 주문 전체를 취소합니다.
  • IOC(Immediate-Or-Cancel)은 체결 가능한 수량만큼 체결하고, 체결되지 않은 잔량은 취소합니다.
    예) 50주 주문 시 40주만 체결 가능한 경우, 40주를 체결하고 10주는 취소합니다.
옵션행사가와 I.V의 상관관계
  • 콜옵션의 경우 행사가가 낮아질수록, 풋옵션의 경우 행사가가 높아질수록 I.V(내재변동성)이 증가합니다.
    그 이유는 옵션 만기 시 행사 가능성이 높은 옵션일수록 KOSPI200과 같은 기초자산의 가격 변동이 일어날 때 상대적으로 변동성이 더 커지기 때문입니다.

선물/옵션 주문 설정 기능

  1. 1
    초기가격 설정
    • 체크 후 가격 설정 시, 주문가격(1-1번)에 초기 설정 가격이 자동 입력됩니다.
    • ‘실시간’ 체크 시 현재가 변화에 따라 실시간으로 주문가격 입력값이 자동 변경됩니다.
  2. 2
    초기수량 설정
    • 체크 후 주문수량 설정 시, 주문수량(2-1번)에 초기 설정 수량이 자동 입력됩니다.
  3. 3
    소리설정
    • 주문 접수가 완료되거나 주문 오류 발생 시 소리로 알려주는 알림음을 설정할 수 있습니다.
  1. 4
    주문 상세설정
    • 주문확인창 : 체크 시 착오주문 방지를 위해 주문 실행 전 확인창을 표시합니다.
    • Enter Key 입력시 주문 실행 : 체크 시 가격 입력창에서 엔터(Enter) 키 입력 시 즉시 주문이 접수됩니다.
    • 탭키이동 : 탭 키로 이동할 때 ‘총가능’, ‘신규’, ‘청산’ 등 주문수량을 지정하는 버튼의 포함 여부를 선택합니다.
  2. 5
    빠른 수량 버튼 설정
    • 클릭 시 주문수량이 자동으로 누적 입력되는 버튼(5-1번)을 7개까지 설정할 수 있습니다.
  3. 6
    주문접수 후 초기화 설정
    • 주문후 커서위치 : 주문 후 커서가 이동할 곳을 지정합니다.
    • 주문후 수량 / 가격지움 : 체크 후 주문 시 입력한 수량 또는 가격을 주문 실행 후 유지하지 않고 삭제합니다.

    • 매도/매수/정정 주문후 원주문 자동입력 :

      체크 시 정정/취소 주문 화면을 조회하면 최근 매도/매수/정정 주문 번호가 자동 입력됩니다.

  4. 7
    경고창 설정
    • 매수/매도 주문 시 설정한 조건에 부합할 경우 주문 전 경고창을 표시해 주문 실수를 방지하도록 도와줍니다.
  5. 8
    주문 가격 실시간 입력 설정
    • 가격 자동체크 안보이기 : 체크 시 주문 화면의 실시간 가격 자동입력 체크박스를 숨깁니다.
