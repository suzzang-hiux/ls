[5400] 주식복수종목주문

늘어나는 관심종목,
주문은 한 번에 간단하게!

시간을 절약하는 가장 효율적인 주문

여러 종목을 빠르게 매수 · 매도하거나 한꺼번에 정정 · 취소해야 할 때, 일일이 한 종목씩 입력해가며 주문하기엔 시간이 부족하죠.

[5400] 주식복수종목주문 화면은 여러 종목의 주문 조건을 한 번에 설정하고 동시에 주문할 수 있어, 시장이 급변하거나 장 마감 직전 신속하게 대응해야 할 때 효과적입니다.

이 화면이 강력한 이유는?

기능 특징
매매 조건 일괄 설정 여러 종목의 주문 유형, 수량, 단가 등을 동시에 설정하고 주문
관심종목 / 보유잔고 연동 주시하는 다수의 종목을 일시에 주문으로 연결
스톱지정가(가격 감시) 지원 특정 가격 도달 시 지정가로 주문 자동 전송

01기능 소개

일괄 매수/매도 주문

  1. 1
    종목명 클릭시 화면 연동
    • 체크 시 목록에서 선택한 종목 정보가 다른 화면에도 연동됩니다.
  2. 2
    호가창 보기 / 숨기기
    • 선택한 종목의 호가창을 우측에 표시하거나 숨길 수 있습니다.
  3. 3
    주문확인창 보기
    • 체크 시 ‘일괄주문‘ 버튼을 클릭하면 주문 접수 전 확인창(3-1번)이 표시됩니다.
  4. 4
    보유잔고/관심종목 연동
    • 잔고수신 버튼 : 클릭 시 보유 잔고 종목이 목록에 추가됩니다.
    • 관심종목 버튼 : 클릭 시 선택한 관심그룹의 종목이 목록에 추가됩니다.
  5. 5
    일괄주문
    • 선택한 종목의 매수/매도주문을 실행합니다.
  6. 6
    주문확인창 보기
    • 매매기준잔고 : 선택 시 [6102] 주식실시간잔고 화면으로 연결됩니다.
    • 체결미체결 : 선택 시 [6104] 주식실시간 체결 / 미체결 화면으로 연결됩니다.
  7. 7
    공통 주문 조건 설정
    • 체크박스에서 선택한 종목에 공통으로 적용할 주문 조건을 설정합니다.
    번호 기능 특징
    7-1 결제방법 현금, 신용, 대출, 대주 중 결제방법 선택
    7-2 거래소 KRX, NXT 중 선택 (스마트 주문 불가)
    7-3 주문유형 보통, 시장가, 중간가 등 선택
    7-4 주문조건 없음, FOK, IOC 중 선택
    7-5 주문수량 직접입력, 금액기준, 비율(매도) 등 입력
    7-6 주문단가 현재가, 전일종가대비(%) 등 입력
  8. 8
    조건선택
    • 일괄주문 조건과 종목 목록을 등록, 편집하거나 불러올 수 있습니다.
FOK / IOC 주문조건이란?
  • FOK(Fill-Or-Kill)은 모든 수량을 체결할 수 있을 때만 주문을 체결하고, 단 1주라도 체결할 수 없으면 주문 전체를 취소합니다.
    예) 50주 주문 시 40주만 체결 가능한 경우, 50주 주문 전체를 취소합니다.
  • IOC(Immediate-Or-Cancel)은 체결 가능한 수량만큼 체결하고, 체결되지 않은 잔량은 취소합니다.
    예) 50주 주문 시 40주만 체결 가능한 경우, 40주를 체결하고 10주는 취소합니다.

일괄 정정/취소 주문

  1. 1
    Clear(초기화)
    • 목록에 표시되는 주문 현황을 모두 삭제합니다. (주문은 취소되지 않습니다.)
  2. 2
    조회
    • 거래소에 따라 시세 및 주문 방식이 달라집니다.
  3. 3
    일괄정정(F4)
    • 버튼 또는 단축키(F4)로 주문확인창(3-1번)에서 선택한 종목의 정정 주문을 실행합니다.
  4. 4
    일괄취소(F5)
    • 버튼 또는 단축키(F5)로 주문확인창에서 선택한 종목의 취소 주문을 실행합니다.
  5. 5
    체결 현황 조회
    • 매매기준잔고 : 선택 시 [6102] 주식실시간잔고 화면으로 연결됩니다.
    • 체결미체결 : 선택 시 [6104] 주식실시간 체결 / 미체결 화면으로 연결됩니다.
  6. 6
    공통 정정 주문 조건 설정
    • 체크박스에서 선택한 종목에 공통으로 적용할 정정 주문 조건을 설정합니다.
    • 매수/매도 주문 중 선택한 종류에 일괄 적용됩니다.

02기타 기능

일괄주문 조건 관리

주문 조건과 종목 목록을 미리 설정해두면 주문 과정이 더 간결해집니다.

  1. 1
    조건 추가
    • 조건 및 종목 목록을 설정하고 저장(1번) 버튼을 클릭합니다.
    • 신규생성(1-1번) 선택 후 조건명을 입력하고, ‘추가’를 선택(1-2번)하여 새 조건을 저장합니다.
  2. 2
    조건 조회
    • 리스트박스에서 저장한 조건을 선택해 조건 및 종목 목록을 조회 후 주문할 수 있습니다.
  3. 3
    조건 수정
    • 조건변경 저장(3-1번) : 조건, 목록 변경사항을 저장합니다.
    • 조건명 + 조건변경 저장(3-2번) : (3-1번) 변경사항 외에 조건명도 수정할 수 있습니다.
  4. 4
    조건 삭제
    • 클릭 시 선택한 조건을 삭제합니다.

스톱지정가 주문

스톱지정가는 현재가가 감시 가격에 도달하면 미리 지정한 가격에 주문을 실행하는 자동 주문의 일종으로, 변동성이 큰 시장 상황에서 이익을 보전하거나 손실을 줄이기 위해 사용하는 전략입니다.

예를 들어 볼까요?

현재가 6,000원인 종목이 6,500원으로 상승 시 6,700원에 매매하려는 경우

  1. 1
    주문유형 선택
    • ‘유형‘ 항목에서 ‘스톱지정가‘ 를 선택합니다.
  2. 2
    주문단가 입력
    • ‘주문단가’ 항목에 매매 희망가인 6,700원을 입력합니다.
  3. 3
    정정 완료
    • ‘조건단가‘ 항목에 현재가 도달 가격인 6,500원을 입력합니다.

👉 조건 설정 후 주문을 접수하면, 현재가가 6,500원에 도달할 경우 매도 주문이 실행되어, 주가 상승 시 일정 이익을 보전하는 전략을 구현할 수 있습니다.

[5400] 주식복수종목주문 화면으로 한 번에 여러 종목을 빠르게 주문하는 방법을 알아보았습니다.

복수종목 주문을 활용하면 여러 종목의 매매 타이밍을 놓치지 않고 깔끔하게 주문을 실행할 수 있습니다.
‘종목명 클릭시 화면 연동‘ 기능을 활용해 종목 정보를 더 치밀하게 분석하고 매매 효과를 높여 보세요.
