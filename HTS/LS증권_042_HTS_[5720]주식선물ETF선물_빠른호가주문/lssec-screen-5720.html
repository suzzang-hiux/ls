호가 클릭만으로 매수 · 매도 주문을 즉시 실행할 수 있으며, 가격 · 체결 · 잔량 변화를 한눈에 보며 주문과 정정을 동시에 처리하는 등 조회와 입력 과정을 최소화하여 빠른 대응과 정확한 체결에 초점을 맞춘 것이 특징입니다.

또한 STOP 주문, 일괄 주문, 잔고 연동 등 주식 선물과 ETF선물 운용에 필요한 전략적 기능도 화면에 포함되어, 시장 흐름을 따라가면서 포지션을 수월하게 관리할 수 있도록 구성하였습니다.

이번 도움말에서는 주요 주문 기능과 효율을 높여주는 화면 설정 방법을 살펴보겠습니다.

  2. 2

    • 종목을 차례대로 조회할 수 있습니다.
  3. 3
    주식선물/ETF선물 종목 정보
    • 종목별 시세, 이론가, 베이시스, 괴리율 등 각 지표를 조회할 수 있습니다.
  4. 4
    체결 정보
    • 체결 시간에 따른 체결가 및 체결수량 추이를 확인할 수 있습니다.
주의하세요!
  • [5720] 주식선물/ETF선물 빠른호가주문 에서는 주식선물, ETF선물을 거래할 수 있으며, 스프레드(SP) 종목, 지수선물옵션, 상품선물은 거래를 지원하지 않습니다.
  • 지수선물옵션, 상품선물은 [5888] 선옵빠른호가주문 에서 주문할 수 있습니다.
  1. 5
    수량 선택
    • 수량을 직접 입력하거나, 빠른 수량 버튼(5-1번)으로 사전에 주문수량을 미리 설정할 수 있습니다.
    • 빠른 수량 버튼(5-1번)은 ’설정 - 주문수량 편집’ 에서 최대 8개까지 지정할 수 있습니다.
  2. 6
    주문가능수량 설정
    • 현재가 또는 기준가 선택 후 매도 · 매수 버튼(6-1번) 클릭 시, 주문가능수량 정보 팝업(6-2번)에서 청산가능, 신규가능, 주문가능 수량을 확인 및 선택하여 주문수량(1번)을 입력할 수 있습니다.
    • ‘신규’(6-3번) 체크 시 주문가능수량 정보 팝업(6-2번) 없이 신규 주문가능수량이 즉시 주문수량(1번)에 입력됩니다.
  3. 7
    주문가능수량 기준 금액 설정
    • 예탁자산 또는 금액을 직접 입력하여 주문가능수량을 계산하는 기준 금액을 설정할 수 있습니다.
    • 금액지정 버튼(7-1번)은 '설정 - 주문수량 편집' 에서 최대 8개까지 지정할 수 있습니다.
선물/옵션에서 사용하는 가격 지표 소개

  • 현재가 :

    시장에서 거래되는 종목의 실시간 가격으로, 매매 시점의 시장 가치를 확인할 수 있는 기본 지표입니다.

  • 이론가 :

    종목을 가격결정모형을 적용해 계산한 이론적인 가격으로, 실제 시장가격과 비교하여 고평가 · 저평가 여부를 판단할 수 있습니다.

  • 만기가 :

    해당 상품의 만기 시점에 정산 및 결제에 사용되는 확정 가격으로, 옵션의 경우 KOSPI200 최종결제지수, 선물의 경우 최종 결제가격이 적용됩니다.

베이시스와 괴리율이란

베이시스와 괴리율은 현물지수(예 : 코스피)와 선물가격 사이의 차이를 분석할 때 사용하는 지표로, 프로그램 매매에서 고평가 · 저평가 상태를 판단하고 차익거래의 기회를 찾는 데 활용됩니다.

  • 베이시스 :

    시장에서 실제로 거래되는 선물가격과 이론적으로 계산된 선물가격의 차이를 의미합니다.
    베이시스가 증가하면(+) 선물 매수가 증가하고, 감소하면(-) 선물 매도가 증가할 가능성이 높습니다.

  • 괴리율 :

    실제 시장의 베이시스가 이론적으로 계산된 베이시스의 값과 얼마나 다른지 백분율(%)로 계산한 수치입니다. 괴리율이 (+)인 경우 선물 고평가 상태, (-)인 경우 선물 저평가 상태입니다. 또 괴리율이 0에 가까워질수록 시장이 균형 상태에 가까워지고 차익거래 압력이 완화되는 것으로 해석할 수 있습니다.


예를 들어 현물 330.00, 선물 331.00인 경우 베이시스는 +1.00p, 괴리율은 +0.30%가 됩니다.
이 경우 선물이 고평가되고 현물이 저평가된 상태이므로, 일반적으로 현물 매수 · 선물 매도 방향의 차익거래가 나타날 수 있습니다.

👉 드디어 기본적인 설정을 마무리하고, 다음 단계에서 실제로 주문을 진행해 보겠습니다.

시장 흐름이 바뀌기 전에,

02 매수/매도 주문하기

  1. 1
    주문수량 설정
    • 매수/매도하려는 주문수량을 입력합니다.
  2. 2
    매도 클릭 주문
    • 원하는 호가의 좌측 매도주문 영역을 더블클릭하면 주문이 접수됩니다.
    • 주문이 접수되면 매도주문 영역에 해당 호가의 누적 미체결 수량이 표시됩니다.
  3. 3
    매수 클릭 주문
    • 원하는 호가의 우측 매수주문 영역을 더블클릭하면 주문이 접수됩니다.
    • 주문이 접수되면 매수주문 영역에 해당 호가의 누적 미체결 수량이 표시됩니다.
  4. 4
    시장가 클릭 주문
    • 매수시장 또는 매도시장의 주문영역을 더블클릭하면 시장가에 주문이 접수됩니다.
클릭 한 번으로 더 빠르게 주문하는 방법
  • ‘설’(설정) 버튼에서 ’매수/매도 주문확인창 보기‘ 및 ’정정/취소 주문확인창 보기’ 체크를 해제하면 호가창 클릭 시 주문확인창 없이 주문이 즉시 접수됩니다.
  • ‘설’(설정) 버튼에서 ‘원클릭 주문‘을 선택하면, 마우스를 한 번만 클릭하는 원클릭으로 주문을 접수할 수 있습니다.
주의하세요!
  • ‘주문확인창 보기’ 체크를 해제하면 확인 절차를 생략하기 때문에 되돌릴 수 없는 주문 실수가 발생할 수 있습니다. 이 점에 유의하여 신중히 설정해 주세요.

손끝으로 즉시 조정하는,

03 정정 주문하기

예를 들어 볼까요?

매수 주문가를 188,900원에서 189,300원으로 정정하는 경우

  1. 1
    기존 주문을 다른 영역으로 드래그
    • 매수 주문 호가 영역(188,900원) 우측 영역을 클릭한 채 드래그해서 정정 호가(189,300원)으로 이동합니다.
  2. 2
    정정 주문 확인
    • 주문 확인 팝업창에서 정정 수량과 정정 가격을 확인하고, ‘정정주문‘ 버튼을 클릭합니다.
  3. 3
    정정 완료
    • 정정 전 호가 영역(188,900원)의 주문 수량 표시는 사라지고, 정정 후 호가 영역(189,300원)에 주문 수량이 표시됩니다.

정리도 전략처럼 단순하게,

04 취소 주문하기

  1. 1
    주문 수량 드래그하여 취소
    • 주문 호가 영역(189,300원)우측에 표시된 수량을 마우스로 클릭 한 뒤 화면 영역 밖으로 드래그합니다.
  2. 2
    확인창에서 주문 내역 확인
    • 팝업창에서 해당 주문의 취소 정보를 확인한 후 ‘취소주문’ 버튼을 클릭하면 주문이 취소됩니다.
  3. 3
    정정 완료
    • 정정 전 호가 영역(189,300원)의 주문 수량 표시는 사라집니다.
여러 건의 주문을 한 번에 취소하려면?
  • 빠르게 대응해야 하는 시장 상황에서 일괄적으로 주문을 취소하는 기능이 유용합니다.
  • 화면 하단 ‘매도취소’ ‘매수취소’ 버튼으로 모든 매수 주문 또는 모든 매도 주문을 한 번에 취소할 수 있습니다.
  • 화면 하단 ‘일괄취소’ 버튼은 모든 매수 주문과 매도 주문을 한 번에 취소할 수 있습니다.

05 기타 기능

호가 정렬 / 새로고침 기능

  1. 1
    호가 정렬 버튼
    • 호가창에 표시되는 호가 범위가 주요 거래 범위를 많이 벗어난 경우, 현재가가 중앙에 위치하고 매도호가, 매수호가가 상하 균형을 맞춰 표시되도록 정렬할 수 있습니다.
    • ‘새’ 버튼으로 새로고침 시에도 모든 정보를 다시 조회하며 호가 정렬 효과를 얻을 수 있습니다.

호가 단위 이동 기능

  1. 1
    5호가 단위 이동 기능
    • ↑ 버튼 클릭 시 매도호가, ↓ 버튼 클릭 시 매수호가가 5개 단위로 추가 표시됩니다.
  2. 2
    1호가 단위 이동 기능
    • ▲ 버튼 클릭 시 매도호가, ▼ 버튼 클릭 시 매수호가가 1개 단위로 추가 표시됩니다.

호가 고정 기능

  1. 1
    호가 고정
    • 설정 화면에서 ‘호‘ 버튼을 클릭하면 호가가 고정됩니다.
      호가를 고정하면 매도호가와 매수호가가 일정하게 고정되며 호가 위치가 변동되지 않습니다.
주의하세요!
  • 호가 고정 기능을 사용할 경우 호가가격의 위치가 변동하게 되므로, 가격이 급변하는 종목에서는 예상과 다른 가격에 주문이 체결될 수 있습니다. 주문 접수 전 실제 주문가격을 반드시 확인해 주세요.

일괄 청산주문 기능

  1. 1
    일괄 청산주문
    • ‘청‘ 버튼 클릭 시 [5912] 선옵일괄청산주문 화면에서 청산 가능한 잔고를 확인하고 일괄 주문할 수 있습니다.

STOP 주문

STOP 주문은 현재가를 기준으로 사용자가 지정한 포착가에 도달하면 원하는 호가에 주문을 접수하는 조건 주문으로, 변동성이 심한 파생상품 시장에서 전략적으로 이익을 실현하거나 손실을 보전할 때 유용합니다.

  1. 1
    STOP 주문 컬럼 추가
    • STOP 주문을 활성화하려면 설정 화면의 ‘STOP 주문 보기’ 에 체크해 주세요.
    • STOP 주문이 활성화되면 매수주문과 매도주문 호가창 측면에 STOP매수(1-2번)와 STOP매도(1-3번) 영역이 추가됩니다.
  2. 2
    STOP 주문 호가 단위(틱) 설정
    • STOP 주문 영역의 호가(포착가) 기준으로, 몇 호가 차이에 주문을 접수할지 지정할 수 있습니다.
    매도 주문 시
    설정 값 주문 실행 방식
    +1 포착가보다 1호가 낮은 가격에 매도 주문 실행 → 즉시 매도주문 실행
    -1 포착가보다 1호가 높은 가격에 매도 주문 실행 → 희망가에 근접해 실행
    매수 주문 시
    설정 값 주문 실행 방식
    +1 포착가보다 1호가 높은 가격에 매수 주문 실행 → 즉시 매수주문 실행
    -1 포착가보다 1호가 낮은 가격에 매수 주문 실행 → 희망가에 근접해 실행
  3. 3
    STOP 주문 일괄취소
    • STOP 매도 / STOP 매수 영역 하단의 ‘STOP취소’ 버튼 클릭 시, 매도 / 매수 STOP 주문 전체가 확인창 없이 일괄 취소됩니다.
    • ‘일괄취소’ 버튼 선택 시 일반 매도/매수 주문을 포함한 STOP 매도 / STOP 매수 주문 전체가 확인창 없이 일괄 취소됩니다.
  4. 4
    STOP 주문 실행 방식
    • 별도의 호가 단위(틱) 설정 없이 STOP 매수 영역(1-1번) 또는 매도 영역(1-2번)에서 매도주문 / 매수주문 영역으로 드래그하면, 원하는 호가 단위(틱)을 자유롭게 조절해서 STOP 주문을 설정할 수 있습니다.
    • STOP 매수 영역(1-1번) 또는 STOP 매도 영역(1-2번)을 더블클릭해도 호가 단위(2번) 설정값만큼 STOP 주문을 설정할 수 있습니다.
    • 현재가가 STOP 주문 영역에 설정한 포착가에 도달하면 매수 / 매도 주문 영역에서 화살표가 가리키는 위치의 호가로 실제 주문이 접수됩니다.

    4-1 예를 들어 볼까요?

    현재가가 190,500원에서 190,800원이 되었을 때 191,100원에 5주 매도하는 주문을 하려는 경우

    • 방법 1 :

      수량 입력란에 5를 입력하고, 포착가(190,800원)의 STOP 매도 영역(1-1번)을 마우스로 클릭한 상태로 호가(191,100원)의 매도주문 영역으로 드래그합니다.

    • 방법 2 :

      수량 입력란에 5를 입력하고, STOP 매도 호가 단위(2번) 숫자를 포착가(190,800원)보다 3호가 위인 ‘-3’으로 설정한 뒤, 호가(190,800원)의 STOP 매도 영역(1-1번)을 더블클릭합니다.

    4-2 예를 들어 볼까요?

    현재가가 190,500원에서 189,600원이 되었을 때 189,700원에 2주 매수하는 주문을 하려는 경우

    • 방법 1 :

      수량 입력란에 2를 입력하고, 포착가(189,600원)의 STOP 매수 영역(1-2번)을 마우스로 클릭한 상태로 호가(189,700원)의 매수주문 영역으로 드래그합니다.

    • 방법 2 :

      수량 입력란에 2를 입력하고, STOP 매수 호가 단위(2번) 숫자를 포착가(원)보다 1호가 위인 ‘1’로 설정한 뒤, 호가(189,600원)의 STOP 매수 영역(1-2번)을 더블클릭합니다.

주의하세요!
  • 설정한 STOP 주문은 서버에 저장되지 않습니다.
    따라서 아래 중 하나에 해당하는 경우 모든 주문이 자동으로 취소되고 조건이 감시되지 않으니 유의해 주세요.
    1) 계좌 또는 종목을 변경하는 경우
    2) 주문 화면을 닫는 경우
    3) HTS를 종료 또는 재시작하는 경우
  • STOP 주문은 주문 조건을 설정하는 자동주문의 일종이며, STOP 주문을 설정하는 즉시 매수/매도 주문이 직접 실행되는 것이 아닙니다. 현재가가 STOP 주문에서 설정한 포착가에 도달하면 실제로 주문이 실행됩니다.
  • 장 시작 전 동시호가 시간대에 STOP 주문 조건을 설정할 경우, 전일 종가와 당일 시가 간의 갭을 시세 포착으로 판단하고 조건이 충족된 것으로 간주되어 주문이 처리될 수 있으니 유의해 주시기 바랍니다.
  • 현재가보다 높은 호가에서 매수 주문이 실행되도록 설정하거나, 현재가보다 낮은 호가에서 매도 주문이 실행되도록 설정한 경우, 현재가가 포착가에 도달하는 즉시 현재가로 주문이 실행됩니다. 예상과 다른 호가에 주문이 체결될 수 있으므로 STOP 주문 설정 시 호가 단위를 꼼꼼히 확인해 주시기 바랍니다.
  • PC 환경, 통신 회선 상태 등 외부 요인으로 인해 주문이 처리되지 않을 수 있습니다. 반드시 조건 설정 및 주문 접수 후에 주문 내역에서 실제 주문 처리 여부를 확인해 주시기 바랍니다.
  • 시세가 급격히 변동되거나 거래가 급증하는 경우 일시적으로 부하가 발생하여 화면 정보가 올바르게 표시되지 않을 수 있습니다. 이 때에는 재조회(새로고침) 기능을 사용하여 시세, 주문, 잔고를 다시 확인해 주시기 바랍니다.
매매 전략을 세울 때 STOP 주문이 중요한 이유
  • STOP 주문은 일반 주문과 달리 지정한 가격 조건에 도달하기 전까지는 나의 주문 물량이 호가창에 표시되지 않습니다. 시장 참여자들에게 나의 매매 전략을 드러내지 않으며 보다 유리한 가격 조건에 매매하는 전략적인 도구로 활용할 수 있습니다.

빠른호가주문 설정

화면 표시 방법이나 주문 방법을 편의에 맞게 설정할 수 있습니다.

  1. 1
    호가 설정
    • 호가고정 : 매도호가와 매수호가가 일정하게 고정되며 호가 위치가 변동되지 않습니다.
    • 화면 비율 늘리기 : 체크 시 상하비율을 조정할 수 있습니다. 해제 시에는 상하로 매도/매수 호가 수를 늘릴 수 있습니다.
    • 호가 셀 높이 : 호가 단위별 높이를 조정할 수 있습니다.

    • 마우스 휠 :

      호가창에서 마우스 휠 상하 이동 시 한 번에 움직이는 호가 단위를 설정할 수 있습니다.
      기본값은 ‘1’로 휠 한 번에 1호가씩 이동하며, ‘2’로 설정 시 휠 한 번에 2호가씩 이동합니다.

  2. 2
    색상 설정
    • 현재가, 호가배경 : 호가창 내 해당 항목의 색상을 설정할 수 있습니다.
    • 평균매입가 보기 : 보유잔고의 평균 매입가에 해당하는 호가에 지정 색상을 표시합니다.

    • 호가 Marking 표시 :

      최대 5개까지 호가 클릭 시 해당 호가에 지정 색상을 표시합니다.
      또 마킹한 호가에 도달 시 화면 깜박임 효과나 소리로 알려주는 옵션을 추가로 설정할 수 있습니다.

    • 평균매입가(●), 미체결주문가(◆) 도형 표시 :

      평균매입가에 도형(●),미체결주문이 있는 호가에 매수/매도 여부에 따라 빨간색, 파란색 도형(◆)을 표시합니다.

  3. 3
    컬럼 설정
    • STOP 주문 보기 : 조건 충족 시 주문을 접수하기 위한 호가 조건 설정 영역을 추가합니다.
    • 건수 항목 보기 : 호가창에서 ‘건수’ 항목 표시 여부를 설정할 수 있습니다.
    • 등락율 보기 : 호가창에서 ‘등락율’ 항목 표시 여부와 대비 기준을 설정할 수 있습니다.
    • 시고저 보기 : 당일 시가, 고가, 저가 호가 우측에 지정 색상이 메모로 표시됩니다.
  1. 4
    기타 설정
    • 매수/매도 주문확인창 보기 : 매수/매도 주문 접수 전 확인창 표시 여부를 설정할 수 있습니다.
    • 정정/취소 주문확인창 보기 : 정정/취소 주문 접수 전 확인창 표시 여부를 설정할 수 있습니다.
    • 커서이미지 보기 : 호가창에 커서를 올렸을 때 주문 / STOP 주문 툴팁 표시 여부를 설정할 수 있습니다.
    • 주문 거부시 메세지 안보기 : 주문 거부 시 사유에 대한 팝업 표시 여부를 설정할 수 있습니다.

    • 툴팁보기(일봉) :

      화면 좌측 일봉 차트에 커서를 올렸을 때 시가, 고가, 저가, 현재가 등 시세 툴팁 표시 여부를 설정할 수 있습니다.

  2. 5
    주문 설정
    • 호가창을 통한 클릭 주문 방법을 설정할 수 있습니다.
      원클릭은 한 번 클릭, 더블클릭은 빠르게 두 번 클릭합니다.
  3. 6
    잔고 수량 설정
    • 잔고 항목을 주문수량에 입력하는 방법을 설정할 수 있습니다.
      원클릭은 한 번 클릭, 더블클릭은 빠르게 두 번 클릭합니다.
  4. 7
    종목, 계좌 변경시 설정
    • 종목 또는 계좌 변경 시 주문수량 입력란에 설정되는 수량을 설정할 수 있습니다.
  1. 8
    키보드 주문기능 및 일괄주문 설정
    • 일반 주문 및 일괄 주문을 빠르게 접수할 수 있도록 키보드 단축키를 직접 지정할 수 있습니다.
  2. 9
    주문수량 설정
    • 주문수량 편집 : 클릭 시 해당 수량이 자동으로 주문수량에 입력되는 버튼을 최대 8개까지 설정할 수 있습니다.

    • 가능수량조회 금액 편집 :

      클릭 시 해당 금액에 해당하는 주문수량이 자동으로 주문수량에 입력되는 버튼을 최대 7개까지 설정할 수 있습니다.
