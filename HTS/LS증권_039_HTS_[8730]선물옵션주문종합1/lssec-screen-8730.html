옵션 민감도란?

옵션 민감도(Greeks)란 시장 상황의 변화에 따라 옵션가격이 민감하게 움직이는 정도를 수치화한 값으로, 옵션의 미래를 예측하는 분석지표로 활용합니다.
옵션 민감도에는 델타, 감마, 베가, 세타, 로 등이 있습니다.

  • 델타(Delta) : 기초자산 가격 변화에 따른 옵션가격의 변화 정도
  • 감마(Gamma) : 기초자산 가격 변화에 따른 옵션 델타의 변화 정도
  • 베가(Vega) : 기초자산 가격의 변동성 1%p 변동에 대한 옵션가격의 변화 정도
  • 세타(Theta) : 잔존일수 1일 경과에 대한 옵션가격의 변화 정도
  • 로(Rho) : 금리의 1%p 변화에 따른 옵션가격의 변화 정도. 지표가 옵션 가격에 미치는 영향은 적은 편
베이시스와 괴리율이란?

베이시스와 괴리율은 현물지수(예 : 코스피)와 선물가격 사이의 차이를 분석할 때 사용하는 지표로,
프로그램 매매에서 고평가 · 저평가 상태를 판단하고 차익거래의 기회를 찾는 데 활용됩니다.

  • 베이시스 :

    시장에서 실제로 거래되는 선물가격과 이론적으로 계산된 선물가격의 차이를 의미합니다.
    베이시스가 증가하면(+) 선물 매수가 증가하고, 감소하면(-) 선물 매도가 증가할 가능성이 높습니다.

  • 괴리율 :

    실제 시장의 베이시스가 이론적으로 계산된 베이시스의 값과 얼마나 다른지 백분율(%)로 계산한 수치입니다. 괴리율이 (+)인 경우 선물 고평가 상태, (-)인 경우 선물 저평가 상태입니다. 또 괴리율이 0에 가까워질수록 시장이 균형 상태에 가까워지고 차익거래 압력이 완화되는 것으로 해석할 수 있습니다.


예를 들어 현물 330.00, 선물 331.00인 경우 베이시스는 +1.00p, 괴리율은 +0.30%가 됩니다.
이 경우 선물이 고평가되고 현물이 저평가된 상태이므로, 일반적으로 현물 매수 · 선물 매도 방향의 차익거래가 나타날 수 있습니다.

05 주문 실행 (D영역)

매수 · 매도 · 정정 · 취소

매수/매도 주문과 정정/취소 주문 기능을 통합하여 신속하게 주문을 관리하고, 단축키로 주문을 빠르게 접수할 수 있습니다.

  1. 1
    주문 버튼 및 연동 기능
    • 매수/매도/정정취소 버튼이 나란히 배치되어 있으며, 입력한 조건에 따라 즉시 주문을 실행할 수 있습니다.
    • 우측의 잔고 미체(미체결), 설정 버튼을 통해 관련 기능 화면으로 빠르게 전환할 수 있습니다.
  2. 2
    종목 선택
    • (2-1번) 선물 / 옵션 전환 : 동일 만기일의 선물(F)과 옵션(O) 종목을 전환할 수 있습니다.
    • (2-2번) 콜옵션 / 풋옵션 전환 : 클릭 시 동일 종목의 콜옵션(C)과 풋옵션(P) 종목을 전환할 수 있습니다.
    • (2-3번) 선물 / 옵션 상품 돌려보기 :
      • 1) 조회 종목이 선물인 경우, 만기일 및 상품을 차례로 전환하며 호가를 조회할 수 있습니다.
      • 2) 조회 종목이 옵션인 경우, 해당 만기일의 행사가를 차례로 전환하며 호가를 조회할 수 있습니다.
  3. 3
    주문유형 선택
    • 유형번호를 직접 입력하거나 목록에서 주문유형을 선택할 수 있습니다.
  4. 4
    주문 수량 입력
    • 수량을 직접 입력하거나 '수' 버튼을 클릭하여 표시되는 팝업에서 설정된 수량을 선택할 수 있습니다.
    • ‘총가능’ 버튼 클릭 시 매수가능한 수량이 입력됩니다.
    • ‘신규’ 클릭 시 신규 주문 가능한 계약수가, '청산' 클릭 시 청산 가능한 보유잔고가 입력됩니다.
    • 수량 입력 버튼(4-1번) 클릭 시 해당 수량이 누적 입력됩니다.
      (예 : 매수 주문창에서 수량 '5' 와 '20' 버튼 클릭 시 '25'가 입력)
  5. 5
    주문가격 입력
    • 가격을 직접 입력하거나 ‘호’ 버튼을 클릭하여 표시되는 호가창에서 선택할 수 있습니다.
    • ‘실시간’ (5-1번) 체크 시 종목의 현재가 변동에 따라 주문가격이 실시간으로 변경됩니다.
    • 입력되는 실시간 가격의 기준은 '설정' 버튼에서 변경할 수 있습니다.
  6. 6
    매수 주문 / 매도 주문 버튼
    • 직접 클릭하거나 단축키로 매수 주문(F2) 또는 매도 주문(F3)을 할 수 있습니다.
  7. 7
    주문 취소 버튼
    • 매수/매도 화면에서도 버튼이나 단축키로 빠르게 미체결 주문을 취소(F5) 할 수 있습니다.
  8. 8
    주문 자동 취소
    • ‘취소연동’ 체크 시, 주문 후 일정 시간 이내에 주문이 체결되지 않으면 주문이 자동으로 취소됩니다.

지정가 지정한 가격으로 주문
시장가 주문단가를 지정하지 않고 시장에서 형성되는 가격으로 자동 지정
조건부 장마감까지 지정가에 주문이 체결되지 않을 경우, 시장가로 주문가격이 전환되어 체결
최유리지정가 상대 1차 호가에 지정
FOK / IOC 주문조건이란?
  • FOK(Fill-Or-Kill)은 모든 수량을 체결할 수 있을 때만 주문을 체결하고, 단 1주라도 체결할 수 없으면 주문 전체를 취소합니다.
    예) 50주 주문 시 40주만 체결 가능한 경우, 50주 주문 전체를 취소합니다.
  • IOC(Immediate-Or-Cancel)은 체결 가능한 수량만큼 체결하고, 체결되지 않은 잔량은 취소합니다.
    예) 50주 주문 시 40주만 체결 가능한 경우, 40주를 체결하고 10주는 취소합니다.
