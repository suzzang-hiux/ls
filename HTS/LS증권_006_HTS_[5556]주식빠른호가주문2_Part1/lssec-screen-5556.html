주문은 빠르게, 판단은 정확하게! 주문과 동시에 시장을 읽다

LS증권 HTS 투혼에서는 더욱 빠르고 직관적인 주식 거래를 위해 [5555] 주식빠른호가주문과 [5556] 주식빠른호가주문2 두 가지 화면을 제공합니다.

[5555] 주식빠른호가주문 화면은 매매에 초점을 맞춰 핵심 기능에 집중했으며, [5556] 주식빠른호가주문2 화면은 차트, 실시간 종목 뉴스 등 다양한 부가 기능을 제공하여, 빠른 주문과 동시에 시장 상황을 파악할 수 있도록 편의성을 높인 것이 특징입니다.

Part 1 시간에는 [5556] 주식빠른호가주문2 화면의 주문 방법을 살펴보고 Part 2 시간에는 주문 편의성을 높여주는 화면설정을 다룰 예정입니다.

빠르고 간편한 주문을 위한

01주문 전 준비

  1. 1
    종목 선택
    • 주문할 종목을 선택합니다.
  2. 2
    거래소 선택
    • 거래소에 따라 시세 및 주문 방식이 달라집니다.
    기능 특징
    스마트 통합시세(KRX + NXT) 제공, 스마트 주문 제출
    (스마트는 당사가 정한 최선집행기준에 따라 KRX와 NXT 중 한 곳으로 주문 제출)
    KRX KRX 시세 제공, KRX 주문 제출
    NXT NXT 시세 제공, NXT 주문 제출
    (NXT 선정 종목만 시세 조회 및 거래 가능)
  3. 3
    선택 종목 잔고 현황
    • 선택한 종목의 보유 잔고 현황을 조회합니다. 보유 잔고가 없으면 빈 칸으로 표시됩니다.
    • (3-1번) 보유수량을 조회합니다. 클릭하면 주문수량 입력란에 보유수량 값이 입력됩니다.
    • (3-2번) 클릭하면 BEP단가 및 평균매입가를 전환하며 조회할 수 있습니다.
    • (3-3번) 클릭하면 수익률 및 평가손익을 전환하며 조회할 수 있습니다.
  4. 4
    주문수량 사전 입력
    • 호가창 클릭 시 사전에 입력해둔 수량으로 바로 주문할 수 있도록 준비합니다.
    • (4-1번) 주문수량을 입력합니다.
    • (4-2번) 3개의 빠른 주문수량 버튼이며 버튼 클릭 시 주문수량에 해당 값이 입력됩니다.
      ‘설정’ 버튼 - ‘빠른 주문수량 입력’에서 주 / % / 금액 입력 단위를 편집할 수 있습니다.
매도/매수 주문 수량 입력란을 통합하거나 분리하고 싶다면

주문수량 설정에서 매도/매수 주문 수량 입력란을 통합할지, 분리할지 선택할 수 있습니다.

  1. 1

    ‘설정’ 버튼을 클릭하여 호가주문 설정을 진행합니다.

  2. 2-1

    한 가지 주문 수량으로 매수/매도 주문을 하려면, 설정 화면에서 ‘주문가능 수량‘을 선택해 주세요.

  3. 2-2

    매수/매도 주문 수량을 각각 설정하려면, 설정 화면에서 ‘매도시 실시간 수량 계산’을 선택해 주세요.

    매도 주문 입력란 ‘자동‘ :

    체크 시 실시간 잔고 기준 ‘총 매도가능 수량 x 설정 비율(%)’이 입력됩니다.

    매도 주문 입력란 % : 총 매도가능 수량의 설정 비율만큼 매도 수량이 입력됩니다.

    매수 주문 입력란 % : 총 매수가능 수량의 설정 비율만큼 매수 수량이 입력됩니다.

한 번 클릭으로 실행하는 주문

02매수/매도 주문하기

  1. 1
    주문수량 확인
    • 주문하려는 수량을 확인합니다.
  2. 2
    호가 클릭으로 주문 실행
    • 매도 주문 : 원하는 호가의 좌측 매도 영역(2-1번)을 더블클릭하면 주문이 실행됩니다.
    • 매수 주문 : 원하는 호가의 우측 매수 영역(2-2번)을 더블클릭하면 주문이 실행됩니다.
클릭 한 번으로 더 빠르게 주문하는 방법
  1. 1

    ‘설정’ 버튼을 클릭한 뒤 설정 화면에서 ‘원클릭 주문‘을 선택하면, 마우스를 한 번만 클릭하는 원클릭으로 주문을 실행할 수 있습니다.

  2. 2

    ‘주문확인창 보기‘ 체크를 해제하면 호가창 클릭 시 주문확인창 없이 주문이 즉시 실행됩니다.

주의하세요!
  • ‘주문확인창 보기’ 체크를 해제하면 확인 절차를 생략하기 때문에 되돌릴 수 없는 주문 실수가 발생할 수 있습니다. 이 점에 유의하여 신중히 설정해 주세요.

드래그 한 번으로 주문 가격 바꾸는

03정정 주문하기

예를 들어 볼까요?

매수 주문가를 5,040원에서 5,010원으로 정정하는 경우

  1. 1
    기존 주문을 다른 영역으로 드래그
    • 매수 주문 호가 영역(5,040원) 우측 영역을 클릭한 채 드래그해서 정정 호가(5,010원)으로 이동합니다.
  2. 2
    정정 주문 확인
    • 주문 확인 팝업창에서 정정 수량과 정정 가격을 확인하고, ‘정정주문‘ 버튼을 클릭합니다.
  3. 3
    정정 완료
    • 정정 전 호가 영역(5,040원)의 주문 수량 표시는 사라지고, 정정 후 호가 영역(5,010원)에 주문 수량이 표시됩니다.

드래그 한 번으로 취소 완료

04취소 주문하기

  1. 1
    주문 수량 드래그하여 취소
    • 주문 호가 우측에 표시된 수량을 마우스로 클릭 한 뒤 화면 영역 밖으로 드래그하면 됩니다.
  2. 2
    확인창에서 주문 내역 확인
    • 팝업창에서 해당 주문의 취소 정보를 확인한 후 ‘취소주문’ 버튼을 클릭하면 주문이 취소됩니다.
여러 건의 주문을 한 번에 취소하려면?
  • 화면 하단 ‘일괄취소’ 버튼은 모든 매수 주문과 매도 주문을 한 번에 취소할 수 있습니다.
  • 버튼을 누르면 취소주문 확인창에서 원하는 주문만 선택해 취소하거나 제외할 수 있습니다.
  • ‘매수취소’, ‘매도취소‘, ‘일괄취소’ 버튼은 빠르게 대응해야 하는 시장 상황에서 유용합니다.

알아두면 호가 주문이 더 편해지는

05기타 기능

다중호가 매매

주문이 여러 호가에 나눠지도록 설정한 뒤 클릭 한 번에 모두 접수하는 기능입니다.
매수의 경우 선택한 호가부터 아래 방향으로, 매도의 경우 위 방향으로 분할된 주문 호가가 일괄 접수됩니다.

예를 들어 볼까요?

LS증권을 5,000원, 4,995원, 4,990원에 각 1주씩 분할 매수하는 경우

  1. 1
    주문수량 입력
    • 주문하려는 수량을 확인합니다. 예시에서는 주문 건당 1주이므로 ‘1’을 입력합니다.
  2. 2
    다중호가매매 체크
    • 다중 호가매매에 체크합니다.
    • 호가수는 주문건수인 ‘3’을, 가격은 호가 간격이 각 1호가이므로 ‘1’틱을 선택합니다.
    • 매수 주문은 선택한 호가부터 아래 방향으로 접수되므로 가장 상단의 5,000원 호가를 더블클릭합니다.
      (매도 주문은 선택한 호가부터 위 방향으로 접수됩니다.)
  3. 3
    주문 확인
    • 주문 확인 팝업창에서 주문 수량과 가격을 확인하고, ‘확인‘ 버튼을 클릭합니다.
  4. 4
    다중호가 주문 완료
    • 한 번의 주문으로 3건의 주문이 분할 접수된 것을 확인할 수 있습니다.

호가 정렬

호가창에 표시되는 호가 범위가 주요 거래 범위를 많이 벗어난 경우, ‘호가정렬’ 버튼을 클릭하면 현재가가 중앙에 위치하고 매도호가, 매수호가가 상하 균형을 맞춰 표시되도록 합니다.

호가 고정 / 실시간 시세 정보

호가를 고정하면 매도호가와 매수호가가 일정하게 고정되며 호가 위치가 변동되지 않습니다.
호가 고정 시 우상단, 좌하단에 시세 정보가 표시되며 다양한 내용을 조회할 수 있습니다.

  1. 1
    호가 고정
    • 우측의 ‘호가고정‘ 버튼이나, 상단의 ‘고정‘ 버튼을 클릭하면 호가가 고정됩니다.
  2. 2
    시세 정보 전환 버튼
    • 버튼을 클릭하면 체결내역, 시세, 피봇, 차트, 중간가 등 다양한 정보를 전환하며 조회할 수 있습니다.

STOP 주문

STOP 주문은 현재가를 기준으로 사용자가 지정한 가격인 포착가에 도달하면 원하는 호가에 주문을 접수하는 조건 주문으로, 주가가 급변하는 상황에서 전략적으로 이익을 실현하거나 손실을 보전할 때 유용합니다.

  1. 1
    STOP 주문 컬럼 추가
    • STOP 주문을 활성화하려면 우측의 ‘스탑주문 활성’ 에 체크해 주세요.
    • STOP 주문이 활성화되면 매수주문과 매도주문 호가창 측면에 STOP매수와 STOP 매도(1-2번)영역이 추가됩니다.
  2. 2
    STOP 주문 호가 단위(틱) 설정
    • 선택한 호가(포착가) 기준으로 몇 호가 차이로 주문을 실행할지 지정할 수 있습니다.
    매도 주문 시
    설정 값 주문 실행 방식
    +1 포착가보다 1호가 낮은 가격에 매도 주문 실행 → 즉시 매도주문 실행
    -1 포착가보다 1호가 높은 가격에 매도 주문 실행 → 희망가에 근접해 실행
    매수 주문 시
    설정 값 주문 실행 방식
    +1 포착가보다 1호가 높은 가격에 매수 주문 실행 → 즉시 매수주문 실행
    -1 포착가보다 1호가 낮은 가격에 매수 주문 실행 → 희망가에 근접해 실행
  3. 3
    일괄취소
    • STOP 매도나 STOP 매수 영역 상하단의 ‘일괄 취소’버튼을 클릭하면 취소확인창 없이 일괄 취소됩니다.
  4. 4
    STOP 주문 실행 방식
    • 별도의 호가 단위(틱) 설정 없이 STOP 매수/매도 영역(1-2번)에서 매수/매도 주문 영역으로 드래그하면, 원하는 호가 단위(틱)만큼 자유롭게 조절해서 STOP 주문을 설정할 수 있습니다.
    • 호가 단위(틱) 설정 후 호가 옆의 STOP 매수/매도 영역(1-2번)을 더블클릭해도 STOP 주문을 설정할 수 있습니다.
    • 현재가가 설정한 포착가에 도달하면 매수/매도 주문 영역에서 화살표가 가리키는 위치의 호가에 실제 주문이 실행됩니다.

    4-1 예를 들어 볼까요?

    현재가가 5,070원에서 5,040원이 되었을 때 5,030원에 5주를 매수하는 주문을 하려는 경우

    • 방법 1 :

      수량 입력란에 5를 입력하고, 호가(5,040원) STOP 매수 영역을 마우스로 클릭한 상태로 호가(5,030원) 매수 영역으로 드래그합니다.

    • 방법 2 :

      수량 입력란에 5를 입력하고, STOP 주문 호가 단위(2번) 숫자를 포착가(5,040원)보다 1호가 아래인 ‘-1’로 설정한 뒤, 호가(5,040원) 매수 영역을 더블클릭합니다.

    4-2 예를 들어 볼까요?

    현재가가 5,070원에서 5,080원이 되었을 때 5,110원에 2주를 매도하는 주문을 하려는 경우

    • 방법 1 :

      수량 입력란에 2를 입력하고, 호가(5,080원) STOP 매도 영역을 마우스로 클릭한 상태로 호가(5,110원) 매도 영역으로 드래그합니다.

    • 방법 2 :

      수량 입력란에 2를 입력하고, STOP 주문 호가 단위(2번) 숫자를 포착가(5,040원)보다3호가 위인 ‘-3’으로 설정한 뒤, 호가(5,080원) 매도 영역을 더블클릭합니다.

주의하세요!
  • 설정한 STOP 주문은 서버에 저장되지 않습니다.
    따라서 아래 중 하나에 해당하는 경우 모든 주문이 자동으로 취소되고 조건이 감시되지 않으니 유의해 주세요.
    1) 계좌 또는 종목을 변경하는 경우
    2) 주문 화면을 닫는 경우
    3) HTS를 종료 또는 재시작하는 경우
  • STOP 주문은 주문 조건을 설정하는 자동주문의 일종이며, STOP 주문을 설정하는 즉시 매수/매도 주문이 직접 실행되는 것이 아닙니다. 현재가가 STOP 주문에서 설정한 포착가에 도달하면 실제로 주문이 실행됩니다.
  • 장 시작 전 동시호가 시간대에 STOP 주문 조건을 설정할 경우, 전일 종가와 당일 시가 간의 갭을 시세 포착으로 판단하고 조건이 충족된 것으로 간주되어 주문이 처리될 수 있으니 유의해 주시기 바랍니다.
  • 현재가보다 높은 호가에서 매수 주문이 실행되도록 설정하거나, 현재가보다 낮은 호가에서 매도 주문이 실행되도록 설정한 경우, 현재가가 포착가에 도달하는 즉시 현재가로 주문이 실행됩니다. 예상과 다른 호가에 주문이 체결될 수 있으므로 STOP 주문 설정 시 호가 단위를 꼼꼼히 확인해 주시기 바랍니다.
  • PC 환경, 통신 회선 상태 등 외부 요인으로 인해 주문이 처리되지 않을 수 있습니다. 반드시 조건 설정 및 주문 접수 후에 주문 내역에서 실제 주문 처리 여부를 확인해 주시기 바랍니다.
  • 시세가 급격히 변동되거나 거래가 급증하는 경우 일시적으로 부하가 발생하여 화면 정보가 올바르게 표시되지 않을 수 있습니다. 이 때에는 재조회(새로고침) 기능을 사용하여 시세, 주문, 잔고를 다시 확인해 주시기 바랍니다.
매매 전략을 세울 때 STOP 주문이 중요한 이유
  • STOP 주문은 일반 주문과 달리 지정한 가격 조건에 도달하기 전까지는 나의 주문 물량이 호가창에 표시되지 않습니다. 시장 참여자들에게 나의 매매 전략을 드러내지 않으며 보다 유리한 가격 조건에 매매하는 전략적인 도구로 활용할 수 있습니다.
