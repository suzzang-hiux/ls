STOP 매수 · 매도

STOP 매수 · 매도는 손절매를 위한 조건 주문 중 하나입니다.
시장가격이 설정한 조건가격에 도달하면 조건주문이 시장가 또는 지정가로 전환되어 주문이 접수됩니다.

  1. 1
    종목 선택
    • 원하는 종목을 직접 입력하여 검색하거나, 검색 아이콘을 클릭하여 팝업에서 상품군 분류에 따라 선택할 수 있습니다.
    • (1-1번) B는 기초자산(Base) 상품군에서 원하는 상품을 선택할 수 있습니다.
    • (1-2번) M은 조회중인 상품의 만기월물(Month)을 선택할 수 있습니다.
  2. 2
    주문유형 선택
    • Market 주문, Limit 주문 중 선택할 수 있습니다.
    분류 설명
    STOP Market 주문
    • 현재 가격보다 불리한 조건가격으로 주문을 접수한 뒤, 시장가격이 조건가격에 도달 시 시장가에 주문이 접수됩니다.
    • 일반 주문은 불리한 조건가격으로 주문 접수 시 시장가에 즉시 주문이 체결되지만, STOP Market 주문은 조건가격에 도달하기 전까지 주문이 접수 및 체결되지 않습니다.

      1) 매수주문 설정 규칙 : 조건가격 > 직전체결가(시장가격)

      2) 매도주문 설정 규칙 : 조건가격 < 직전체결가(시장가격)
    STOP Limit 주문
    • STOP Market 주문과 동일하게 현재 가격보다 불리한 조건가격으로 주문을 접수한 뒤, 시장가격이 조건가격에 도달 시 지정한 가격에 주문이 접수됩니다.
    • STOP Market은 별도의 가격 설정 없이 시장가에 주문이 접수되지만, STOP Limit은 설정한 주문가격으로 주문이 접수됩니다.

      1) 매수주문 설정 규칙 : 주문가격>=조건가격>직전체결가(시장가격)

      2) 매도주문 설정 규칙 : 주문가격<=조건가격<직전체결가(시장가격)
  3. 3
    주문 수량 입력
    • 수량을 직접 입력하거나 ‘수’ 버튼을 클릭하여 팝업에서 수량을 선택할 수 있습니다.
    • ‘신규’ 클릭 시 신규 주문 가능한 계약수가, ‘청산’ 클릭 시 청산 가능한 보유잔고가 입력됩니다.
    • ‘총가능’ 버튼 클릭 시 신규 및 청산 가능한 총 주문가능 수량이 입력됩니다.
    • 수량 입력 버튼(4-1번) 클릭 시 해당 수량이 누적 입력됩니다.
      (예 : 매수 주문창에서 수량 '5' 와 '20' 버튼 클릭 시 '25' 입력)
  4. 4
    주문가격 입력
    • 가격을 직접 입력할 수 있습니다.
  5. 5
    조건가격 입력
    • STOP 주문 조건에 사용되는 가격을 입력합니다.
  6. 6
    매수 주문 / 매도 주문
    • 직접 클릭하거나 단축키로 매수 주문(F2) 또는 매도 주문(F3)을 할 수 있습니다.
  7. 7
    설정
    • 주문확인창, 수량 입력 버튼, 주문 후 초기화 등 주문 관련 설정을 할 수 있습니다.
주의하세요!
  • 해외선물옵션 Stop 주문은 네트워크 장애, 시세지연, 천재지변 등 다양한 요인에 의해 정상적으로 작동하지 않을 수 있으니 반드시 사용법과 유의사항을 숙지하신 후 이용하여 주시기 바랍니다.
  • CME 거래소를 포함한 일부 해외거래소의 STOP 주문은 시장가격이 조건가격에 도달했을 시 시장가 주문이 아닌 거래소 자체에서 지정가로 접수됩니다.
    이로 인해 시장이 급등락하거나 호가 공백이 발생하는 등의 상황에서 체결이 되지 않거나, STOP 가격보다 불리한 가격에 체결될 가능성이 있으니 유의하시기 바랍니다.

OCO 매수 · 매도

OCO(One Calcels Other) 주문은 지정가 주문, STOP 시장가 주문 두 가지의 주문을 동시에 접수한 뒤 이 중 하나가 체결되면 나머지 하나는 자동으로 취소되는 해외선물 주문 방법입니다.

  1. 1
    종목 선택
    • 원하는 종목을 직접 입력하여 검색하거나, 검색 아이콘을 클릭하여 팝업에서 상품군 분류에 따라 선택할 수 있습니다.
    • (1-1번) B는 기초자산(Base) 상품군에서 원하는 상품을 선택할 수 있습니다.
    • (1-2번) M은 조회중인 상품의 만기월물(Month)을 선택할 수 있습니다.
  2. 2
    주문 수량 입력
    • 가격을 직접 입력하거나 ‘수’ 버튼을 클릭하여 팝업에서 수량을 선택할 수 있습니다.
    • ‘신규’ 클릭 시 신규 주문 가능한 계약수가, '청산' 클릭 시 청산 가능한 보유잔고가 입력됩니다.
    • ‘총가능’ 버튼 클릭 시 신규 및 청산 가능한 총 주문가능 수량이 입력됩니다.
    • 수량 입력 버튼(4-1번) 클릭 시 해당 수량이 누적 입력됩니다.
      (예 : 매수 주문창에서 수량 '5' 와 '20' 버튼 클릭 시 '25' 입력)
  3. 3
    지정가 입력
    • 매수 / 매도 주문 가격을 직접 입력할 수 있습니다.
  4. 4
    조건가격 입력
    • STOP 주문 조건에 사용되는 가격을 입력합니다.
    • 1) OCO 매수 주문의 경우 : 지정가 < 현재가 < STOP 가격(조건가) 요건이 되도록 가격을 설정합니다.
    • 2) OCO 매도 주문의 경우 : STOP 가격(조건가) < 현재가 < 지정가 요건이 되도록 가격을 설정합니다.
    • ※ STOP 가격과 지정가는 최소 20틱 이상의 차이가 있어야 합니다.
  5. 5
    매수 주문 / 매도 주문
    • 직접 클릭하거나 단축키로 매수 주문(F2) 또는 매도 주문(F3)을 할 수 있습니다.
    • 주문 접수 후 아래 가격 도달 요건에 따라 주문이 다르게 처리됩니다.
    • 1) 지정가 주문가격 도달 후, 지정가 주문이 전량 체결 시 : STOP 주문은 취소됩니다.
    • 2) 지정가 주문가격 도달 후, 지정가 주문이 일부 체결 시 : 미체결 잔량만큼 지정가 주문 및 STOP 주문이 모두 유효합니다.
    • 3) STOP 조건가격 도달 시 : 미체결 상태의 지정가 주문 잔량은 취소되고, 미체결 잔량만큼 시장가에 주문이 접수됩니다.
  6. 6
    설정
    • 주문확인창, 수량 입력 버튼, 주문 후 초기화 등 주문 관련 설정을 할 수 있습니다.

    예를 들어 볼까요?

    (신규 주문)

    현재가 100일 때, 90에 신규 매수하고 싶으나, 만약 110 이상 오른다면 바로 매수하기를 원할 때

    • → 지정가 90, Stop 가격(조건가) 110으로 'OCO 매수' 1계약 집행
      (증거금은 1계약 해당액만큼만 필요)
    • → 90과 110 중 먼저 도달하는 쪽으로 매수 체결, 나머지 하나는 소멸됨

    예를 들어 볼까요?

    (청산 주문)

    매도 1계약 보유 중이며 현재가 100일 때, 90에 청산하여 이익을 실현하고 싶으나, 만약 110 이상 오른다면 바로 청산하여 손절하기를 원할 때

    • → 지정가 90, Stop 가격(조건가) 110으로 'OCO 매수' 1계약 집행
      (청산주문이기 때문에 증거금은 추가로 필요하지 않음)
    • → 90과 110 중 먼저 도달하는 쪽으로 매수 체결, 나머지 하나는 소멸됨
주의하세요!
  • 해외선물옵션 OCO 주문은 네트워크 장애, 시세지연, 천재지변 등 다양한 요인에 의해 정상적으로 작동하지 않을 수 있으니 반드시 사용법과 유의사항을 숙지하신 후 이용하여 주시기 바랍니다.
  • OCO 주문은 부가서비스 주문 형태이며 여러 가지 사유로 주문이 정상적으로 접수되지 않을 수 있습니다.
    OCO 주문을 접수한 뒤에도 수시로 주문 현황 및 잔고를 확인하여 자산의 손실이 발생하지 않도록 유의하셔야 합니다.
  • 아래 유의사항을 숙지하지 않아 발생한 손해에 대해서는 당사에서 책임을 지지 않으니 유의해 주세요.
시세/시스템 관련
  • 당사 해외선물 OCO 주문은 HTS 등 주문매체를 로그아웃(종료)하여도 해당일 장 종료시까지 유효합니다.
  • Stop 가격(조건가) 도달 여부는 현재가(체결가) 데이터가 들어오는 시점을 기준으로 판단하며, 네트워크 장애, 시세 지연, 천재지변 등의 요인에 의해 실제 시세 도달 시점과 시차가 존재할 수 있습니다.
  • Stop 가격(조건가) 도달시에는 당사 서버에서 자동으로 지정가 주문 미체결 잔량을 소멸시키고 그 수량만큼 시장가로 주문을 접수하며, 따라서 이 때 Stop 가격(조건가)와 실제 체결가격은 상이할 수 있습니다. (Stop Market 주문과 동일)
  • 시세가 급변하여 순간적으로 Stop 가격(조건가)에 도달 후 바로 반등 또는 반락하였을 경우 '지정가 주문 소멸 및 Stop(시장가) 주문 집행 절차' 가 이루어지지 않을 수 있습니다. 이 경우 지정가 주문과 Stop(시장가) 주문은 계속 유효하게 됩니다.
  • 시스템 장애로 인한 백업서버 전환 처리 시 주문 중지 또는 원하지 않는 호가에 주문이 처리될 수 있습니다.
  • 시스템 전체 혹은 부분 장애, 특정부분의 오류(운영장비, 프로그램, 시세 등), 시스템 백업 전환 등으로 인하여, 서비스 지속보다 중지가 필요하다고 판단되는 경우, 전체 주문에 관하여 정지시킬 수 있습니다.
주문/조회 관련
  • 본 주문은 시장가 주문을 허용하지 않는 거래소(SGX, HKEX) 상품에 대해서는 실행이 불가합니다.
  • 가격 설정 시 OCO 매매 가격 설정에서 제시한 조건 중 하나라도 충족되지 못할 경우 본 주문은 실행되지 않습니다.
  • OCO 주문은 정정할 수 없으며, 취소만 접수할 수 있습니다. 취소 주문을 접수하면 OCO 지정가, OCO Stop 주문이 일괄 취소되며, 둘 중 하나만 개별 취소하는 것은 불가능합니다.
  • [5965] 해외선물 빠른호가주문 화면의 호가창에서는 OCO주문 조회 및 실행을 할 수 없으며, 동 화면의 '일괄 취소' 기능을 실행해도 OCO주문은 취소되지 않습니다. (단, 주문내역에서는 조회됨)
  • OCO Stop 주문은 현재가(체결가)가 Stop 가격(조건가)에 도달하기 전까지는 '미체결' 내역에 표시되지 않고 '주문내역'에만 표시됩니다.
손해배상 제외 범위
  • 고객의 고의, 과실, 전산기기 조작 오류로 인하여 발생한 손해
  • 회사는 공지의 의무를 다하였으나 고객이 본 서비스 이용에 대한 이해 부족으로 발생하는 손해
  • 회사가 신의성실의 주의 의무를 다하였음에도 부득이하게 발생하는 오류 정보로 인한 손해
  • 회사에 책임이 없는 컴퓨터 범죄로 인하여 발생한 손해
  • 고객의 비밀번호 관리 소홀로 발생한 주문으로 인한 손해
  • 전시, 사변, 천재지변, 기타 불가항력 등으로 인한 전산 및 회선 장애로 발생한 손해
  • 당사와 무관하게 거래소, 해외 FCM 등 대외기관의 주문/체결 지연, 시세 지연, 시세 오류 등으로 인하여 발생한 손해

설정

화면 표시 방법이나 주문 방법을 편의에 맞게 설정할 수 있습니다.

종목 선택
  1. 1

    주문 접수가 완료되거나 주문 오류 발생 시 소리로 알려주는 알림음을 설정할 수 있습니다.

주문 상세설정
  1. 2

    주문확인창 : 체크 시 착오주문 방지를 위해 주문 실행 전 확인창을 표시합니다.

  2. 3

    Enter Key 입력시 주문 실행 : 체크 시 가격 입력창에서 엔터(Enter) 키 입력 시 즉시 주문이 접수됩니다.

  3. 4

    빠른 수량 버튼 설정 : · 클릭 시 주문수량이 자동으로 누적 입력되는 버튼을 7개까지 설정할 수 있습니다.

주문접수 후 초기화 설정
  1. 5

    주문후 커서위치 : 주문 후 커서가 이동할 곳을 지정합니다.

  2. 6

    주문후 수량 / 가격지움 : 체크 후 주문 시 입력한 수량 또는 가격을 주문 실행 후 유지하지 않고 삭제합니다.

경고창 설정
  1. 7

    매수/매도 주문 시 설정한 조건에 부합할 경우 주문 전 경고창을 표시해 주문 실수를 방지하도록
    도와줍니다.
