[5555] 주식빠른호가주문 - Part 2 「화면설정」

매매 결과를 바꾸는
‘화면설정’ 한 끗 차이

주문 속도와 정확도를 높이는 설정 노하우

클릭 한 번이면 주문이 실행되는 LS증권 투혼 HTS [5555] 주식빠른호가주문 방식과 다양한 기능, 지난 콘텐츠에서 빠르게 살펴보았습니다.

이번 시간에는 [5555] 주식빠른호가주문 화면의 다양한 ‘화면설정’ 방법에 대해 알아보겠습니다.

빠른 매매의 완성은 ‘화면설정'에서 시작됩니다.

이 화면이 실전 매매에 도움이 되는 이유!

기능 특징
등락률 기준가 설정 호가 변동의 기준가를 직접 설정해 수익 판단 정확도 향상
강조 색상 설정 호가창에서 현재가, BEP단가, 호가 Marking 표시 구분 가능
주문 실행 방식 설정 매수 · 매도 방식을 익숙한 방식으로 설정 가능
빠른 주문 수량 · 금액 설정 반복 주문을 줄이고, 실수 없이 빠르게 입력 가능
호가 정렬 / 수량 반영 설정 화면 중앙 고정 + 매수 가능 수량 자동 반영으로 효율적 대응
STOP 주문 설정 조건부 예약 주문으로 시장가격이 사전에 설정한 스톱가격에 도달시, 지정한 호가 가격으로 매매거래 참여

[5555]주식빠른호가주문 화면에서 ‘설정’ 버튼을 누르면 다양한 기능을 설정할 수 있는 팝업화면이 보입니다.
이제부터 본격적으로 ‘설정’ 화면을 살펴보겠습니다.

기준은 다르게, 판단은 빠르게

01등락률 기준 바꾸기

  1. 1
    등락률 칼럼 추가
    • 설정 메뉴에서 ‘등락률 표시’ 항목을 선택하면, 호가창에 등락률 칼럼이 새로 생성됩니다. (1-1번)
    • 기준가 옵션은 ‘기준가’, ‘전일종가’, ‘시가’, ‘고가’, ‘저가’, ‘BEP(손익분기점)’, 사용자 정의 가격까지 다양하게 제공됩니다.

눈에 띄게, 클릭은 실수없이

02색상 강조 설정하기

  1. 1
    호가 색상 설정
    • 호가창 구성 항목별로 다양한 색상을 적용할 수 있습니다. (1-1번)
    • 현재가 호가 색상, 호가창 배경색, BEP 단가 색상, 특정 호가 클릭 시 색상 변경 기능을 활용하여 실시간 시세 변화의 흐름을 보다 빠르게 파악할 수 있습니다.

입력은 손에 맞게, 실행은 한 번에

03주문 실행 방식 설정하기

  1. 1
    주문 실행 방식 설정
    • 호가 영역(1-1번)에서 주문 접수 방식을 선택할 수 있습니다.
      더블클릭 주문(기본 설정) : 마우스를 두 번 연속으로 클릭해 주문을 제출합니다.
      원클릭 주문 : 마우스를 한 번만 클릭해 주문을 제출합니다.
  2. 2
    키보드 주문 기능
    • 키보드 단축키를 활용한 주문이 가능합니다.
    항목 단축키
    매수주문 Enter
    매도주문 Enter
    일괄매도 Ctrl + Enter
    정정주문 Shift (선택) → 이동 → Shift + Enter (주문)
    취소주문 Delete
    전량취소 Shift + Delete
    매수/매도 전량취소 Ctrl + Delete
    매도가능 Ctrl + Q
    매수가능 Ctrl + W
    가능금액 Ctrl + A

반복 입력 줄이고, 속도는 높이고

04빠른 수량・금액 설정하기

  1. 1
    빠른수량 설정
    • 클릭하면 주문 수량 입력란에 자동 입력되는 주문 수량 또는 주문 비율을 미리 설정할 수 있습니다.
    • 주식수(주)로 설정한 경우 : 호가 화면에 (1-1번)과 같이 설정한 수량이 반영됩니다.
    • 비율로 설정한 경우 : 호가 화면에 (1-2번)과 같이 설정한 비율이 반영됩니다.
  2. 2
    매수 금액 설정
    • 주문 수량이 아닌 금액 기준으로 사전 설정 가능하며 설정 완료 시 (2-1번)과 같이 설정한 금액이 반영됩니다.
    • 금액 설정 후 호가 화면에서 원하는 호가에 마우스를 올리면 설정된 금액에 맞춰 주문가능 수량이 변동됩니다.
주문 가능 수량을 비율로 설정하는 방법
  • 빠른수량 설정이 활성화되어 있다면, 호가에서 마우스 우측 버튼을 클릭해 매수/매도가능 수량을 전체 가능 수량의 일정 비율로 입력할 수 있습니다.

더 빠른 주문을 도와주는

05호가 정렬 & 수량 반영 설정하기

  1. 1
    호가 가운데 정렬 기능
    • 마우스 스크롤 등으로 이동된 호가창을 현재가 기준으로 다시 가운데 정렬할 수 있습니다.
    • ‘호가 가운데 정렬’ 항목을 체크하고, 호가를 더블 클릭하면 현재가가 화면 중앙으로 자동 정렬 됩니다. (1-1번)
    참고해주세요!
    • 정렬 기능을 사용하려면 ‘호가 고정’(1-2번) 설정을 해제해야 정상 작동합니다.
    • ‘호가 고정’이 체크되어 있으면 가운데 정렬이 실행되지 않습니다.
  2. 2
    주문 가능 수량 자동 반영
    • 호가 영역을 더블 클릭하면, 해당 호가에 주문 가능한 수량(2-1번)이 자동으로 반영됩니다.
    • 수량을 따로 입력할 필요 없이 즉시 주문 가능 상태로 전환됩니다.

종목은 바뀌어도, 기준은 그대로?

06수량 유지・초기화 설정하기

  1. 1
    종목,계좌 변경 시 수량 처리 방식 설정
    • 종목이나 계좌를 바꿀 때, 기존에 조회한 매수 가능 수량(1-1번)을 어떻게 처리할지 설정할 수 있습니다.
    옵션 기능
    초기화 종목이 변경되면 기존 수량은 삭제되고, 새 종목 기준 0부터 다시 조회
    유지 종목이 바뀌어도 기존에 조회된 수량이 그대로 유지됨
    매수 가능수량 조회 변경된 종목에 맞춰 수량이 자동으로 재조회됨

가격에 의미를, 잔량에 힌트를

07호가 메모 & 잔량 대비 보기

  1. 1
    호가메모 보기
    • 당일 시가, 고가, 저가에 해당하는 가격을 호가창에서 색상으로 구분표시(1-1번)해주며, 마우스 오버시 툴팁으로도 내용을 보여줍니다.
    • 당일 시세 변동 상황에서 중요한 가격 포인트를 실시간 확인할 수 있습니다.
    • 설정 버튼을 통해 당일 시가, 고가, 저가 등 구성 항목별로 다양한 색상 적용이 가능합니다. (1-2번)
  2. 2
    잔량대비 보기
    • 호가창에 잔량대비 컬럼을 추가하여 변동되는 잔량 증감 상태를 파악할 수 있습니다. (2-1번)

눈과 손 모두 편하게

08기타 사용자 설정 기능

  1. 1
    신용주문구분 표시
    • 신용 주문일 경우, 해당 호가의 좌측 공란이 빨간색으로 채워집니다.
    • 현금 주문일 경우, 공란으로 유지됩니다.
  1. 2
    커서 이미지 보기
    • 매수/매도 영역을 마우스로 드래그 시, 마우스 커서 색상이 검정색으로 변경되며 툴팁으로 ‘매수주문’ 또는 ‘매도주문’ 안내가 함께 표시됩니다.
  2. 3
    화면 비율 늘리기
    • 화면 비율 늘리기 기능 체크 시, 호가 화면의 크기를 확대 / 축소하여 사용자가 원하는 비율대로 변경 가능합니다.
  3. 4
    시장가 주문 사용 설정
    • ‘시장가 주문 사용’ 체크 시, 해당 영역(4-1번) 클릭만으로 시장가 주문 실행이 가능합니다.
  4. 5
    호가 고정 설정
    • 호가창의 스크롤 이동을 막고, 기준 호가를 (5-1번)과 같이 고정할 수 있습니다.
    • 마우스 드래그 시 화면이 움직이지 않아 집중도를 유지할 수 있습니다.
    • 호가 고정을 선택하면 호가 가운데 정렬 기능은 작동하지 않으며, STOP 주문 시 포착가와 주문가를 표시하는 화살표가 나타나지 않습니다.
  5. 6
    일봉 툴팁 보기
    • 일봉 차트에 마우스를 올리면 시가, 고가, 저가, 현재가 정보가 툴팁으로 실시간 표시됩니다. (6-1번)
  6. 7
    체결량 색상 표시 설정
    • 체결량 조건 또는 거래대금 기준을 설정하고, 조건 충족 시 해당 색상으로 표시되도록 설정 합니다. (7-1번)
  7. 8
    마우스 휠 이동 범위 설정
    • 호가 창에서 마우스 휠 상하 이동시에 한 번에 움직이는 호가 단위를 설정할 수 있습니다.
    • 기본값은 ‘1’ → 휠 한 번에 1호가씩 이동하며, '2'로 설정하면 휠 한 번에 2호가씩 이동합니다.

지금까지 살펴본 설정 항목만 잘 활용해도 매매의 정확도와 속도에서 확실한 차이를 경험할 수 있습니다.
