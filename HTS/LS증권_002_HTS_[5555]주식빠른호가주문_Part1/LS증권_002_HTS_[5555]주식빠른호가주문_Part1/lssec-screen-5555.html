  2. 2

02매수/매도 주문하기

  1. 1
    '매수/매도 가능 수량 조회' 버튼

    • 매도가능 버튼 :

      클릭 시 (1-1번) 수량 입력란에 해당 종목의 매도가능한 수량이 자동으로 입력됩니다.

    • 매수가능 버튼 :

      클릭 시 (1-2번) 매수 가능한 수량을 조회하는 팝업이 표시됩니다.
      요건을 입력하고 '조회' 버튼으로 매수 가능한 수량을 조회한 뒤, '확인' 버튼을 클릭하면 수량 입력란에 주문 수량이 입력됩니다.

  1. 2
    호가 클릭만으로 주문 실행

    • 매수 주문 :

      원하는 호가의 오른쪽 영역(2-1번)을 더블클릭하면 주문이 실행됩니다.

    • 매도 주문 :

      원하는 호가의 왼쪽 영역(2-2번)을 더블클릭하면 주문이 실행됩니다.

    클릭 한 번으로 더 빠르게 주문하는 방법
    1. 1
      ‘설정’ 버튼 클릭 후 설정 화면에서 ‘원클릭 주문‘을 선택하면, 마우스를 한 번만 클릭하는 원클릭으로 주문을 실행할 수 있습니다.
    2. 2
      ‘주문확인창 보기‘ 체크를 해제하면 호가창 클릭 시 주문확인창 없이 주문이 즉시 실행됩니다.
    주의하세요!
    • ‘주문확인창 보기’ 체크를 해제하면 확인 절차를 생략하기 때문에 되돌릴 수 없는 주문 실수가 발생할 수 있습니다. 이 점에 유의해 신중히 설정해 주세요.
  1. 3
    주문 수량 표시
    • 주문이 완료되면 매수호가 및 매도 호가 창에 수량이 표시됩니다.
    • 매수수량표시(3-1번), 매도수량표시(3-2번)
  2. 4
    빠른 수량 선택 가능
    • 4가지의 주문수량을 사전에 입력해두고, 버튼처럼 클릭 한 번에 주문 수량을 즉시 입력할 수 있습니다.
  3. 5
    금액 기준 주문 수량 자동계산
    • '금액주문(5번)'에 체크가 되어 있다면, 주문 금액을 입력한 후 (3-1번) 영역과 (3-2번) 영역에 마우스를 올리고 움직여보세요.
      입력한 주문 금액만큼 매수/매도 수량이 자동 계산되어 주문수량 입력란에 입력됩니다.
주식 주문 유형 정리
  • 호가창 상단의 ▼ 버튼을 클릭하면 (1번)과 같이 주문 유형을 선택할 수 있으며, 1-1영역에서 주문을 실행합니다.
  • 주문 유형

03정정하기

예를 들어 볼까요?

매수 주문가를 58,800원에서 59,100원으로 정정하는 경우

  1. 1
    정정한 주문 수량 클릭 후 드래그
    • 매수주문 호가(58,800원) 클릭 후 마우스 드래그하여 정정 호가(59,100원)로 이동합니다.
  2. 2
    정정 주문 확인
    • 주문 확인 팝업창이 나타나며, 정정 수량과 가격 확인 후 '정정주문' 버튼을 클릭합니다.
  3. 3
    정정 완료
    • 정정 전 호가 가격(58,800원) 영역에 있던 수량 표시는 사라지고, 정정 후 호가 가격(59,100원) 영역에 주문 수량이 표시됩니다.
참고해주세요!
  • ‘주문건별 정정취소’ 버튼 클릭 및 활성화 시, 같은 호가에 접수한 다수의 주문을 다른 호가로 정정하거나 취소할 때 어떤 주문을 정정하거나 취소할지 주문번호를 선택하는 팝업이 표시됩니다.

04취소하기

  1. 1
    주문 수량 드래그하여 취소
    • 주문 호가 우측에 표시된 수량을 마우스로 클릭 한 뒤 화면 영역 밖으로 드래그하면 됩니다.
  2. 2
    확인창에서 주문 내역 확인
    • 팝업창에서 해당 주문의 취소 정보를 확인한 후 ‘취소주문’ 버튼을 클릭하면 주문이 취소됩니다.
여러 개의 주문을 한 번에 취소하려면?
  • 화면 아래의 ‘매수취소’ 버튼이나 ‘매도취소’ 버튼으로 모든 매수 주문 또는 모든 매도 주문을 한 번에 취소할 수 있습니다.
  • 화면 아래의 ‘일괄취소’ 버튼은 모든 매수 주문과 매도 주문을 한 번에 취소할 수 있습니다.
  • 버튼을 누르면 취소주문 확인창에서 원하는 주문만 선택해 취소하거나 제외할 수 있습니다.
  • ‘매수취소’, ‘매도취소‘, ‘일괄취소’ 버튼은 빠르게 대응해야 하는 시장 상황에서 유용합니다.

05기타기능

다중호가 매매 기능

주문이 여러 호가에 나눠지도록 설정한 뒤 클릭 한 번에 모두 접수하는 기능입니다.
매수의 경우 선택한 호가부터 아래 방향으로, 매도의 경우 위 방향으로 분할된 주문 호가가 일괄 접수됩니다.

예를 들어 볼까요?

삼성전자 주식을 20주 보유하고 있는데
60,000원에 10주와 60,300원에 10주 분할 매도를 하고자 할 경우

  1. 1
    다중 주문 실행 체크
    • 여러 호가에 분할매매를 실행하고 싶을 때, 다중 주문을 진행합니다.
      ‘다중’ 좌측 체크박스를 체크합니다.
  2. 2
    동시 주문 건수 설정
    • 몇 건의 주문을 동시에 낼지 선택합니다.
    • 예시에서는 주문이 2건이므로 '2'를 선택합니다.
  3. 3
    호가 간격 설정
    • 각 주문 간의 호가 간격(틱 수)를 지정합니다.
    • 예시에서는 주문하려는 60,000원과 60,300원의 호가 간격이 3호가이므로, 호가간격인 틱(3번)에서 '3'을 선택합니다.
  4. 4
    다중 주문 접수
    • 설정을 완료하고 호가가 60,000원인 곳 좌측의 매도호가 영역에서 주문을 접수합니다.
      그러면 조건에 맞춰 60,000원에 10주, 60,300원에 10주, 이렇게 총 2건의 매도 주문이 일괄 접수된 것을 확인할 수 있습니다.

호가창 위,아래 정렬 기능

  1. 1
    호가 중앙 정렬 기능
    • C 버튼 클릭 시, 노란색의 현재가 가격이 중앙에 위치하며 매도호가, 매수호가가 상하 균형을 맞춰 표시됩니다.
  1. 2
    5호가 단위 이동 기능
    • ↑ 버튼 클릭 시 매도호가는 5개 단위로 추가되어 표시됩니다.
      (↓ 버튼 클릭 시 매수호가 5개 단위로 추가 표시)
  1. 3
    1호가 단위 이동 기능
    • ▲ 버튼 클릭 시, 매도호가는 1개 단위로 추가되어 표시됩니다.
      (▼ 버튼 클릭 시 매수호가 1개 단위 추가 표시)
  1. 4
    호가 고정 기능
    • ‘호’ 버튼 클릭 시, 호가 화면은 매도호가와 매수호가 사이에 라인이 적용되며 호가 개수 변동이 안되는 고정된 화면으로 적용됩니다.

STOP 주문 설정하기

STOP 주문은 현재가를 기준으로 사용자가 지정한 가격인 포착가에 도달하면 원하는 호가에 주문을 접수하는 조건 주문으로, 주가가 급변하는 상황에서 전략적으로 이익을 실현하거나 손실을 보전할 때 유용합니다.

  1. 1
    STOP 주문 컬럼 추가
    • STOP 주문을 활성화하려면 설정 버튼(1번)을 클릭하고 컬럼설정의 STOP 주문 보기(1-1번) 항목을 체크해 주세요.
    • STOP 주문이 활성화되면 매수주문과 매도주문 호가창 측면에 STOP 매수와 STOP 매도(1-2번)영역이 추가됩니다.
  2. 2
    STOP 주문 호가 적용 단위 설정
    • 선택한 호가(포착가) 기준으로 몇 호가 차이로 주문을 실행할지 지정할 수 있습니다.
      매도 주문 시
      설정 값 주문 실행 방식
      +1 포착가보다 1호가 낮은 가격에 매도 주문 실행 → 즉시 매도주문 실행
      -1 포착가보다 1호가 높은 가격에 매도 주문 실행 → 희망가에 근접해 실행
      매수 주문 시
      설정 값 주문 실행 방식
      +1 포착가보다 1호가 높은 가격에 매수 주문 실행 → 즉시 매수주문
      -1 포착가보다 1호가 낮은 가격에 매수 주문 실행 → 희망가에 근접해 실행
  3. 3
    STOP 주문 실행 방식
    • 별도의 호가 단위(틱) 설정 없이 STOP 매수/매도 영역(1-2번)에서 매수/매도 주문 영역으로 드래그하면, 원하는 호가 단위(틱)만큼 자유롭게 조절해서 STOP 주문을 설정할 수 있습니다.
    • 호가 단위(틱) 설정 후 호가 옆의 STOP 매수/매도 영역(1-2번)을 더블클릭해도 STOP 주문을 설정할 수 있습니다.
    • 현재가가 설정한 포착가에 도달하면 매수/매도 주문 영역에서 화살표가 가리키는 위치의 호가에 실제 주문이 실행됩니다.

      3-1 예를 들어 볼까요?

      현재가가 4,990원에서 4,970원이 되었을 때 4,955원에 10주를 매수하는 주문을 하려는 경우

      • 방법 1 :

        수량 입력란에 10을 입력하고, 호가 4,970원 우측의 STOP 매수 영역을 마우스로 클릭한 상태로 호가 4,955원 우측의 매수주문 영역으로 드래그합니다.

      • 방법 2 :

        수량 입력란에 10을 입력하고, STOP 매수(2번)의 숫자를 포착가(4,970원)보다 3호가 아래인 ‘-3’으로 설정한 뒤, 호가 4,970원 우측의 STOP 매수 영역을 더블클릭합니다.

      3-2 예를 들어 볼까요?

      현재가가 4,990원에서 5,010원이 되었을 때 5,020원에 30주를 매도하는 주문을 하려는 경우

      • 방법 1 :

        수량 입력란에 30을 입력하고, 호가 5,010원 좌측의 STOP 매도 영역을 마우스로 클릭한 상태로 호가 5,020원 좌측의 매도주문 영역으로 드래그합니다.

      • 방법 2 :

        수량 입력란에 30을 입력하고, STOP 매도(2번)의 숫자를 포착가(5,910원)보다 1호가 높은 ‘-1’로 설정한 뒤, 호가 5,010원 우측의 STOP 매도 영역을 더블클릭합니다.

  4. 4
    STOP 주문 일괄 취소
    • STOP 매수 또는 STOP 매도 하단의 ‘STOP취소‘ 버튼을 클릭하면 한 쪽의 매수/매도 주문이 취소확인창 없이 일괄 취소됩니다.
    • ‘일괄취소’ 버튼을 클릭하면 모든 STOP 매수와 STOP 매도 주문이 취소확인창 없이 일괄 취소됩니다.
주의하세요!
  • 설정한 STOP 주문은 서버에 저장되지 않습니다.
    따라서 아래 중 하나에 해당하는 경우 모든 주문이 자동으로 취소되고 조건이 감시되지 않으니 유의해 주세요.
    1) 계좌 또는 종목을 변경하는 경우
    2) 주문 화면을 닫는 경우
    3) HTS를 종료 또는 재시작하는 경우
  • STOP 주문은 주문 조건을 설정하는 자동주문의 일종이며, STOP 주문을 설정하는 즉시 매수/매도 주문이 직접 실행되는 것이 아닙니다. 현재가가 STOP 주문에서 설정한 포착가에 도달하면 실제로 주문이 실행됩니다.
  • 장 시작 전 동시호가 시간대에 STOP 주문 조건을 설정할 경우, 전일 종가와 당일 시가 간의 갭을 시세 포착으로 판단하고 조건이 충족된 것으로 간주되어 주문이 처리될 수 있으니 유의해 주시기 바랍니다.
  • 현재가보다 높은 호가에서 매수 주문이 실행되도록 설정하거나, 현재가보다 낮은 호가에서 매도 주문이 실행되도록 설정한 경우, 현재가가 포착가에 도달하는 즉시 현재가로 주문이 실행됩니다. 예상과 다른 호가에 주문이 체결될 수 있으므로 STOP 주문 설정 시 호가 단위를 꼼꼼히 확인해 주시기 바랍니다.
  • PC 환경, 통신 회선 상태 등 외부 요인으로 인해 주문이 처리되지 않을 수 있습니다. 반드시 조건 설정 및 주문 접수 후에 주문 내역에서 실제 주문 처리 여부를 확인해 주시기 바랍니다.
  • 시세가 급격히 변동되거나 거래가 급증하는 경우 일시적으로 부하가 발생하여 화면 정보가 올바르게 표시되지 않을 수 있습니다. 이 때에는 재조회(새로고침) 기능을 사용하여 시세, 주문, 잔고를 다시 확인해 주시기 바랍니다.
매매 전략을 세울 때 STOP 주문이 중요한 이유
  • STOP 주문은 일반 주문과 달리 지정한 가격 조건에 도달하기 전까지는 나의 주문 물량이 호가창에 표시되지 않습니다. 시장 참여자들에게 나의 매매 전략을 드러내지 않으며 보다 유리한 가격 조건에 매매하는 전략적인 도구로 활용할 수 있습니다.
