[5220] 주식스탑로스 - Part 1 「화면소개」

기회를 놓치지 않는,
똑똑한 자동 주문

조건만 걸어두면, 나머지는 알아서 자동으로 착착

스탑로스(Stop Loss) 주문은 “어떤 종목이 오르는 추세를 보일 때 사고 싶어요.”
“보유한 주식의 수익이 일정 %를 달성하면 팔고 싶어요”처럼 주문 조건과 매매 내용을 미리 정해두면, 조건 충족 시 자동으로 매수/매도해주는 기능입니다. 주식 화면을 계속 지켜보지 않아도, 손실은 줄이고 수익은 지키게 도와주는 똑똑한 안전장치죠.

스탑로스 주문은 세 단계로 진행되는데요, 주문을 접수하는 기준을 설정하고, 주문금액과 주문수량을 정한 뒤, 조건 감시를 시작합니다.
그러면 HTS가 실시간으로 시장을 관찰하다가 조건이 충족되는 즉시 자동으로 주문을 접수합니다.

[5220] 주식스탑로스 화면, 어떤 점이 좋을까요?

  1. 1
    계속 차트를 볼 필요가 없어요

    조건만 세팅해두면 자동 주문 완료!
    바쁜 직장인이나 초보 투자자에게 편리해요.

  2. 2
    시장 변화에 대비할 수 있어요

    어느날 갑자기 주가가 요동쳐도, 정해둔 요건과 가격에 자동 매매되니 걱정 뚝.
    큰 손실을 방지할 수 있고, 목표 수익 달성하기도 수월해요.

  3. 3
    수익 날 때는, 바로 팔지 않고 더 오래 들고 가요

    트레일링 스탑(T/S) 주문을 활용하면, 주가가 오를수록 목표가격도 같이 오르니까 주가가 오르면 계속 보유하고, 떨어지면 빠르게 정리하는 유연한 매매가 가능해요.

모든 시장 흐름을 놓치지 않고, 24시간 신속하게 대응합니다

이 화면이 편리한 이유는?

기능 특징
전략별 맞춤 주문조건 설정 간편한 설정부터 고급 설정까지 지원
장기간 조건 감시 지원 최대 90일까지 지속되는 자동 주문 조건
실시간 조건 감시 제어 필요할 때, 필요한 종목만 자동 주문 실행
분할매도 지원 일정 시간 간격으로 분할매도 주문 가능
다른 종목 감시 기능 다른 종목의 시세 변동을 시세포착 조건으로 활용

조회 · 설정 · 감시가 하나의 흐름으로 이어집니다

01화면 구성

화면은 다음 3개 영역으로 나뉘며, 종목을 선택하고 주문 조건 설정을 거쳐 자동으로 조건 감시와 주문을 실행하는 (A) - (B) - (C) 순서에 따라 기능이 배치되어 있습니다.

  1. A
    종목 조회 영역
    • 자동주문 종목을 선택하고 시세를 조회할 수 있습니다.
  2. B
    주문 조건 설정 영역
    • 선택한 종목의 주문 조건을 저장하거나 수정할 수 있습니다.
    항목 특징
    간편매도설정(개별) 보유 종목의 매도 주문 조건을 설정합니다.
    고급매도설정(개별) 다른 종목 시세, T/S 등 매도 주문 조건을 더 세밀하게 설정할 수 있습니다.
    매도공통조건설정 보유 종목을 비롯한 전체 종목에 매도 주문 조건을 동일하게 설정합니다.
    복수매수설정 다수의 종목에 동일한 매수 주문 조건을 적용하여 접수할 수 있습니다. (당일만 유효)
    간편매수설정 원하는 종목의 매수 주문 조건을 설정합니다.
    고급매수설정(B&S) 한 번의 조건 설정으로 매수 주문과 매도 주문을 자동으로 접수합니다.

    • 매수조건 + 매도주문조건 :

      매수 조건 포착→ 매수 주문 접수 → (매수 체결 후)매도 조건 포착 → 매도 주문 접수

    • 즉시매수 + 매도주문조건 :

      매수 주문 접수 → (매수 체결 후)매도 조건 포착 → 매도 주문 접수
    정정/취소 주문을 정정하거나 취소할 수 있습니다.
  3. C
    감시 설정 · 내역 조회 영역
    • 주문 조건 감시를 실행/중지하거나, 주문 내역 및 잔고 등을 확인할 수 있습니다.
    항목 특징
    매도설정내역 설정한 매도 주문 조건을 조회 · 삭제하거나, 조건 감시를 실행/정지할 수 있습니다.
    매수설정내역 설정한 매수 주문 조건을 조회 · 삭제하거나, 조건 감시를 실행/정지할 수 있습니다.
    조건포착내역 감시 실행 중 충족된 주문 조건과 자동 접수된 주문 내역을 조회할 수 있습니다.
    실시간주문체결내역 실제 접수된 주문 내역을 조회하고, 주문을 일괄 정정 또는 일괄 취소할 수 있습니다.
    실시간잔고 실시간 잔고 현황을 조회하고, 선택한 종목을 일괄 주문할 수 있습니다.
    예수금 예수금, 추정인출가능금액, 익일 추정반대매매 내역을 확인할 수 있습니다.

02주식스탑로스 시작하기

주식스탑로스 이용 신청

  1. 1
    서비스 신청
    • ‘신청/해지’ 버튼 클릭 시 주식 스탑로스주문 이용 신청/해지 팝업을 조회합니다.
  2. 2
    계좌 및 신청여부 선택
    • 주식 스탑로스 주문을 이용할 계좌를 선택하고, ‘신청’을 선택합니다.
  3. 3
    기본사항 및 유의사항 확인
    • 주식 스탑로스 주문 이용 시 사용법과 유의사항을 읽고, 확인 및 이용동의(3-1, 3-2, 3-3번)에 체크합니다.
  4. 4
    신청
    • ‘확인’ 버튼 클릭 시 서비스 신청이 완료됩니다.

체결통보 신청

‘체결통보신청’(1번) 버튼 클릭 시 [7707] 똑똑알리미 화면에서 주문 체결 시 알림 받을 매체를 문자(PUSH/SMS) 또는 E-mail 중 선택(2번)할 수 있습니다.

탐색은 빠르게, 대응도 빠르게

03종목 조회 (A영역)

시세 정보 조회 및 주문 조건 연결

  1. 1
    종목 선택 및 조회
    • 자동 주문할 종목을 선택하고, 시세를 조회할 수 있습니다.
  2. 2
    호가 입력
    • 현재가(2-1번), 5호가 영역(2-2번), 시가, 고가, 저가, 상한가, 하한가(2-3번) 클릭 시, 주문 조건 설정 영역에서 가격 입력란에 선택한 값을 입력할 수 있습니다.

04주문 조건 설정 (B영역)

시세 포착 조건 · 주문 실행 조건 설정

  1. 1
    시세 포착 조건 설정
    • 주문을 자동 접수하는 시점을 결정하는 시세 포착 조건을 설정합니다.
  2. 2
    주문 실행 조건 설정
    • (1번)에서 설정한 시세 포착 조건이 충족된 경우, 자동으로 접수하는 주문 내용을 입력합니다.
  3. 3
    주문 조건 유효기간 및 확인 방식 설정
    • (3-1번) 주문 조건의 유효기간을 설정합니다.
    • (3-2번) 시세 포착 조건 충족 시 주문확인창 여부를 선택합니다.

      1) 자동 : 조건 충족 시 주문확인창 없이 즉시 자동으로 주문을 접수합니다.

      2) 수동 : 조건 충족 시 주문확인창을 표시합니다.

  4. 4
    주문 조건 저장 및 수정
    • ‘조건신규저장’ 버튼 클릭 시 새 주문 조건을 저장합니다.
    • ‘조건수정’ 버튼으로 기존 조건 변경사항을 저장할 수 있습니다.
참고해주세요!
  • 조건 등록 가능 시간 : 07:20 ~ 익일 04:59
  • 주문 조건은 매수 주문 조건, 매도 주문 조건 각 최대 100건까지 설정할 수 있습니다.
  • 주문 조건의 유효기간은 최대 90일까지 설정할 수 있습니다.
  • 주문확인방식이 ‘수동‘인 경우, HTS가 실행되지 않은 상태에서 시세 포착 조건 충족 시 주문이 접수되지 않으며 알림도 별도로 발송되지 않습니다.

05감시 설정 · 내역 조회 (C영역)

시세 포착 조건 감시 실행 및 정지 · 내역 조회

  1. 1
    조회 조건 설정
    • 조회를 원하는 주문 조건을 설정할 수 있습니다.
  2. 2
    감시 실행 버튼

    • 1) 개별 종목 감시 실행 :

      ‘OFF’ 또는 ‘ON’ 버튼 클릭 시 시세 포착 조건 감시 비활성화 상태(OFF)와 감시 활성화 상태(ON)(2-1번)를 전환할 수 있습니다.

    • 2) 전 종목 일괄 감시 실행 :

      ‘전체’ 버튼(2-2번) 클릭 시 모든 주문 조건을 감시 활성화 상태(ON)으로 전환합니다.

    • 3) 전 종목 일괄 감시 정지 :

      ‘매수감시 전체정지’ 버튼(2-3번) 또는 매도설정내역 탭 → ‘매도감시 전체정지‘ 버튼 클릭 시 모든 주문 조건을 감시 비활성화 상태(OFF)로 전환합니다.

  3. 3
    삭제
    • ‘x’ 버튼 클릭 시 해당 조건을 삭제합니다.
    • ‘전체’ 버튼(3-1번) 클릭 시 모든 조건을 삭제합니다.
주의하세요!
  • 시세 포착 조건 감시가 활성화 상태(ON)이고 주문 확인 방식이 ‘자동’ 이면 HTS를 종료하거나 PC를 종료하더라도 조건 충족 시 주문이 자동으로 접수됩니다.

  • 시세 포착 및 자동주문 접수 시간 : 09:00 ~ 15:20

주문 내역 · 실시간 잔고 · 예수금 조회

  • ‘조건포착내역’ 및 ‘실시간주문체결내역’ 화면에서 조건을 포착하고 실제 주문 접수와 체결로 이어진 내역을 확인할 수 있습니다.
  • ‘실시간잔고’ 및 ‘예수금’ 화면에서 주문 체결에 따른 잔고와 예수금의 변동사항을 실시간으로 조회할 수 있습니다.
